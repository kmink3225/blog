## Getting Started

### Prerequisites

- [Quarto](https://quarto.org/docs/get-started/) installed
- Python 3.11+ (for code execution)
- R (optional, for R-based examples)

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/kmink3225/blog.git
cd blog
```

2. Install Python dependencies:
```bash
pip install -r blog_requirements.txt
```

3. Preview the site:
```bash
quarto preview
```

4. Build the site:
```bash
quarto render
```

The rendered site will be in the `_site` directory.
