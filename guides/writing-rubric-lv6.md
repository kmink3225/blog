# Lv.6 채점 루브릭 — 실무 보고서 / 전문가 칼럼

> **핵심 목표**: 의사결정권자를 설득하고 실질적인 행동 변화를 유도하는가  
> **합격선**: R1 Gate Pass + 44/50  

---

## R10 — Rhetoric (수사의도)

> **Lv.6 기준**: 독자(의사결정권자)의 심리적 저항을 낮추고 설득을 유도하는가?

| 점수 | 기준 |
|------|------|
| 5 | 독자의 페인 포인트를 관통하는 수사적 장치를 배치하여 설득을 완성함 |
| 3 | 정보 전달은 완벽하나 '움직이게 만드는' 힘은 부족함 |
| 1 | 정보 나열에 그쳐 설득력이 없음 |

---

## R11 — Diction (언어 통제 / 전문적 절제)

> **Lv.6 기준**: 군더더기 없는 절제된 언어로 핵심을 타격하는가?

| 점수 | 기준 |
|------|------|
| 5 | 불필요한 단어 하나 없이 정교하게 가공된 언어로 전문성을 극대화함 |
| 3 | 문장이 명확하나 일부 장황하거나 불필요한 수식어가 있음 |
| 1 | 문장이 장황하고 어휘 선택이 모호하여 전문성이 떨어짐 |
