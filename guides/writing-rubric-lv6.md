# Lv.6 채점 루브릭 — 실무 보고서

> **핵심 목표**: Executive summary를 갖추고 의사결정자가 바로 행동할 수 있는 근거를 제시하는가  
> **합격선**: R1 Gate Pass + 38/50 (75% 고정 — 모든 레벨 공통)  
> **설계 원칙**: 모든 기준의 1~5점은 실무 보고서 수준 글의 스펙트럼 안에서 정의된다.  
> Lv.5와의 핵심 차이: 논증의 정교함보다 의사결정자 관점의 명확성이 우선된다. 결론이 서두에 있고, 데이터가 비즈니스 임팩트로 번역되며, 독자가 전체를 읽지 않아도 핵심을 파악할 수 있어야 한다.

---

## BLUF 구조 참조

| 요소 | 설명 |
|------|------|
| Executive Summary | 핵심 결론 + Recommendation — 서두에 배치 |
| 근거·분석 | 데이터·사실 기반 논거 — 비즈니스 임팩트로 번역 |
| 리스크·제약 | 반론·한계·조건 — 의사결정 리스크로 제시 |
| Action Items | 구체적 실행 방안 — 누가 언제 무엇을 |

---

## R1 — Stance [Gate]

| 판정 | 기준 |
|------|------|
| **Pass** | 핵심 결론과 Recommendation이 서두(Executive summary)에 명확히 제시됨 |
| **Soft Fail** | 결론은 있으나 서두가 아닌 끝에 배치되거나, Recommendation 없이 분석만 있음 |
| **Hard Fail** | 결론·Recommendation 없음, 또는 목적 불명확 |

---

## R2 — Focus (주제 일관성)

> **평가 질문**: 모든 섹션이 Executive summary의 Recommendation을 뒷받침하는가?

| 점수 | 기준 |
|------|------|
| 5 | 모든 섹션이 Recommendation을 직접 뒷받침. 불필요한 배경 설명 없음 |
| 4 | 한 섹션이 Recommendation과 약하게 연결되나 전체 방향 유지 |
| 3 | 배경 설명이 과도하여 핵심 메시지가 희석되나 의도는 파악됨 |
| 2 | 두 섹션 이상이 Recommendation과 무관하거나 방향이 엇갈림 |
| 1 | 절반 이상이 Recommendation과 무관 |

---

## R3 — Syntax (문법/언어 정확성)

> **평가 범위**: 간결하고 명확한 실무 문장, 전문 용어 정확 사용, 불필요한 수식어 배제  
> **Lv.6 기준**: 장황한 표현이나 학술체 잔재는 3점 이하. 전문 용어 오용은 2점 이하

| 점수 | 기준 |
|------|------|
| 5 | 간결하고 명확한 실무 문장, 전문 용어 정확 사용, 불필요한 수식어 없음 |
| 4 | 대체로 간결하나 한두 곳 불필요하게 복잡한 표현이 있음 |
| 3 | 문장이 장황하거나 학술체 표현이 남아 실무 보고서 어조에 맞지 않음 |
| 2 | 장황한 표현이 빈번하거나 전문 용어를 오용하여 신뢰도가 떨어짐 |
| 1 | 의미 파악이 어려울 정도의 오류 또는 완전한 비전문적 표현 |

---

## R4 — Cohesion (유기성/연결성)

> **평가 질문**: Executive summary와 본문 각 섹션이 논리적으로 연결되는가?  
> **Lv.6 기준**: Executive summary가 본문 내용을 정확히 요약하지 못하면 2점 이하

| 점수 | 기준 |
|------|------|
| 5 | Executive summary → 근거·분석 → 리스크 → Action items가 논리적으로 연결. 각 섹션이 다음을 자연스럽게 예고 |
| 4 | 전체 흐름 자연스러우나 한 섹션에서 Executive summary와의 연결이 약함 |
| 3 | 섹션 간 전환은 있으나 Executive summary와 본문 간 연결이 피상적 |
| 2 | 섹션들이 독립적으로 존재하거나 Executive summary가 본문 내용을 제대로 반영하지 못함 |
| 1 | 논리적 흐름 없음 |

---

## R5 — Structure (논리 전개 구조)

> **Lv.6 요구 구조**: BLUF — Executive summary → 근거·분석 → 리스크·제약 → Action items  
> **평가 질문**: 독자가 Executive summary만 읽어도 의사결정이 가능한가?

| 점수 | 기준 |
|------|------|
| 5 | BLUF 완성. Executive summary(결론+Recommendation) → 근거·분석 → 리스크·제약 → Action items 갖춰짐. Executive summary만으로 의사결정 가능 |
| 4 | BLUF 구조 있으나 Action items가 약하거나 리스크·제약 분석이 부족 |
| 3 | 결론이 앞에 있고 근거가 뒤따르나 Action items 없음 |
| 2 | 기본 구조는 있으나 BLUF 미적용 — 결론이 끝에 배치됨 (Lv.5 이하 구조) |
| 1 | 구조 없음, 생각의 흐름대로 나열 |

---

## R6 — Depth (논거 독립성/심화)

> **Lv.6 기준**: 데이터·수치를 비즈니스 임팩트로 번역해야 함. 수치 나열만으로는 3점  
> **평가 질문**: "이 숫자가 우리 비즈니스에 어떤 의미인가?"를 독자에게 전달하는가?

| 점수 | 기준 |
|------|------|
| 5 | 논거가 구체적 데이터 기반이고, 수치·결과를 비즈니스 임팩트로 번역하여 의사결정자가 바로 활용 가능 |
| 4 | 구체적 데이터 기반 논거 + 일부 임팩트 번역, 나머지는 수치만 나열 |
| 3 | 구체적 데이터 기반 논거, 임팩트 번역 없이 수치만 제시 (Lv.5 수준에 그침) |
| 2 | 논거 있으나 데이터 없이 일반적 주장 수준 |
| 1 | 논거 없음, 주장만 반복 |

---

## R7 — Flow (가독성/독자 경험)

> **평가 질문**: 바쁜 의사결정자가 빠르게 핵심을 파악할 수 있는가?  
> **Lv.6 기준**: 전체를 읽어야만 핵심 파악이 가능하면 3점 이하

| 점수 | 기준 |
|------|------|
| 5 | 바쁜 의사결정자가 Executive summary만으로 전체 논지와 결론을 파악 가능. 본문은 필요 시 참조 |
| 4 | 핵심 파악은 가능하나 2~3곳에서 추가 설명을 읽어야 의미가 완성됨 |
| 3 | 전체를 읽어야 핵심 파악 가능, 중간에 멈추게 됨 |
| 2 | 전반적으로 핵심 메시지를 찾기 어렵고 구조를 재파악해야 함 |
| 1 | 읽는 내내 핵심 메시지 파악 불가 |

---

## R8 — Paraphrasing (표현 다양성 / 재진술)

> **Lv.6 적용 범위**: B+A+C 완숙 + 실무 맥락 재구성  
> **평가 질문**: 데이터·연구·외부 주장을 의사결정자 언어로 재구성하여 전달하는가?

| 점수 | 기준 |
|------|------|
| 5 | 데이터·연구·외부 주장을 실무 맥락에 맞게 정확히 재구성하여 의사결정자 언어로 자연스럽게 전달 |
| 4 | 외부 내용 재구성 시도하나 학술적 표현이 남아 실무 맥락에 완전히 맞지 않음 |
| 3 | 외부 내용 재진술은 있으나 원문 표현에 의존하거나 실무 맥락으로의 변환 없음 |
| 2 | 외부 내용을 직접 인용에만 의존하거나 핵심 표현 반복 |
| 1 | 주장·논거를 그대로 복사·반복 |

---

## R9 — Universality (논거·근거의 보편성)

> **Lv.6 기준**: 해당 비즈니스·산업 맥락에 직접 적용 가능한 근거여야 함  
> **평가 질문**: 인용한 데이터가 이 상황의 의사결정에 실제로 적용 가능한가?

| 점수 | 기준 |
|------|------|
| 5 | 논거가 해당 비즈니스·산업 맥락에 직접 적용 가능한 데이터·사례에 기반하고, 적용 범위와 한계도 실무적으로 명시 |
| 4 | 관련 산업 데이터 기반이나 적용 맥락의 차이나 한계 언급 없음 |
| 3 | 일반적 연구·데이터 기반, 해당 비즈니스 맥락 적합성 검토 없음 (Lv.5 수준에 그침) |
| 2 | 사회·역사적 사례 중심, 비즈니스 데이터 없음 |
| 1 | 개인 경험·감정만, 보편화 시도 없음 |

---

## R10 — Rhetoric (수사 효과)

> **Lv.6 적용 범위**: 의사결정자의 행동을 유도하는 수사 전략 평가  
> **평가 질문**: Urgency·강조·대조가 Executive에게 행동을 촉발하도록 배치되는가?

| 점수 | 기준 |
|------|------|
| 5 | Executive summary 첫 문장이 의사결정 임팩트를 직접 제시하고, urgency·대조·강조가 행동 유도에 전략적으로 배치됨 |
| 4 | 임팩트 제시와 수사 장치가 효과적이나 한 곳에서 설득력이 약함 |
| 3 | 정보 전달은 명확하나 의사결정자의 행동을 촉발하는 장치가 약함 |
| 2 | 평이한 정보 나열, urgency·대조 없음 |
| 1 | 수사 전략 부재, 의사결정자 관점 미반영 |

---

## R11 — Diction (언어 경제성)

> **Lv.6 적용 범위**: 비즈니스 terminology의 정확·경제적 구사. 장황함은 치명적  
> **평가 질문**: Executive가 한 번에 핵심을 파악할 정도로 단어 하나하나가 압축적인가?

| 점수 | 기준 |
|------|------|
| 5 | 비즈니스 terminology를 정확·경제적으로 구사하고 긴 설명을 한 단어로 압축. 잉여 수식 없음 |
| 4 | 대체로 경제적이나 한두 곳에서 장황한 풀이가 남음 |
| 3 | 장황한 풀이가 곳곳에 남아 Executive가 핵심을 파악하는 데 노력이 필요 |
| 2 | 잉여 수식·장황함이 지배적이어서 핵심이 묻힘 |
| 1 | 의사결정자가 단시간에 핵심을 파악할 수 없을 정도의 장황함 또는 부정확한 용어 사용 |
