# Lv.2 채점 루브릭 — 고등학생

> **핵심 목표**: 근거를 제시하고 논리 비약을 인식할 수 있는가  
> **합격선**: R1 Gate Pass + 20/40  
> **설계 원칙**: 모든 기준의 1~5점은 고등학생 글의 스펙트럼 안에서 정의된다.  
> Lv.1과의 핵심 차이: 주장이 한 문장으로 정리되어야 하고, Reasoning이 추가되며, 개인 경험만으로는 보편성 기준을 충족하지 못한다.

---

## R1 — Stance [Gate]

| 판정 | 기준 |
|------|------|
| **Pass** | 주장이 한 문장으로 정리되고 찬성/반대 입장이 명확함 |
| **Soft Fail** | 입장은 있으나 "~인 것 같다", "~일 수도 있다" 식으로 모호하거나 단정 짓지 못함 |
| **Hard Fail** | 주장이 없거나 주제와 전혀 무관한 내용 |

---

## R2 — Focus (주제 일관성)

> **평가 질문**: 모든 문단이 thesis를 향해 기여하는가?

| 점수 | 기준 |
|------|------|
| 5 | 모든 문단이 thesis에 직접 기여, 불필요한 문단 없음 |
| 4 | 한 문단이 thesis와 약하게 연결되나 전체 흐름 유지 |
| 3 | 한 문단이 주제와 느슨하게 연결되어 집중도가 떨어지나 의도는 파악됨 |
| 2 | 두 문단 이상이 주제에서 벗어나거나 관련 없는 내용을 포함 |
| 1 | 글의 절반 이상이 주제와 무관 |

---

## R3 — Syntax (문법/언어 정확성)

> **평가 범위**: 기초 문법은 기본, 문장 구조의 다양성과 어휘 선택까지 확장  
> **Lv.2 기준**: 단순 문장만 반복하면 3점 이하

| 점수 | 기준 |
|------|------|
| 5 | 문법 오류 없음, 단순·복합 문장을 혼용하며 어휘 선택이 적절함 |
| 4 | 오류 1~2개, 문장 구조가 다소 단조롭지만 의미 전달에 지장 없음 |
| 3 | 오류 3~4개이거나 문장 구조가 전체적으로 단순 반복됨 |
| 2 | 오류가 빈번하거나 어휘 선택이 반복적으로 어색함 |
| 1 | 기초 문법 오류가 심해 문장 의미 파악이 어려움 |

---

## R4 — Cohesion (유기성/연결성)

> **평가 질문**: 문단 간 전환이 자연스러운가?  
> **Lv.2 기준**: 단순 연결어("다음으로", "또한") 사용은 3점, 앞 문단 내용이 다음 문단을 예고하면 5점

| 점수 | 기준 |
|------|------|
| 5 | 문단 간 전환이 자연스럽고, 앞 문단의 마지막 문장이 다음 문단을 자연스럽게 예고함 |
| 4 | 연결어를 사용하여 전환하나 한 곳에서 약간 어색한 점프가 있음 |
| 3 | 연결어를 사용하나 문단 간 논리 연결이 표면적("다음으로", "또한" 수준에 그침) |
| 2 | 문단 간 전환이 갑작스러워 독자가 맥락을 스스로 재파악해야 함 |
| 1 | 문단들이 독립적으로 존재하며 서로 연결이 없음 |

---

## R5 — Structure (논리 전개 구조)

> **Lv.2 요구 구조**: Claim(주장) → Reasoning(이유 설명) → Example(예시) 3단계  
> **평가 질문**: 주장이 왜 맞는지 이유를 설명한 후 예시로 뒷받침하는가?

| 점수 | 기준 |
|------|------|
| 5 | 주장 → 이유 설명 → 구체적 예시 → 마무리가 모두 갖춰지고 각 단계가 명확히 연결됨 |
| 4 | 3단계 구조가 있으나 한 단계가 약하거나 간략하게만 처리됨 |
| 3 | 주장 + 예시는 있으나 이유 설명(Reasoning)이 빠져 있음 |
| 2 | 주장과 예시 중 하나만 명확히 있음 |
| 1 | 구조 없음, 생각의 흐름대로 나열 |

---

## R6 — Depth (논거 독립성/심화)

> **Lv.2 기준**: 논거 1개도 가능하나 구체성이 있어야 3점. 서로 다른 관점의 논거 2개면 5점

| 점수 | 기준 |
|------|------|
| 5 | 서로 다른 관점의 논거 2개 이상 (예: 개인적 측면 + 사회적 측면) |
| 4 | 논거 2개이나 같은 관점을 반복한 수준 |
| 3 | 논거 1개, 주장과 명확히 연결되고 구체적 설명이 있음 |
| 2 | 논거 1개이나 "그게 나쁘니까" 수준으로 주장과의 연결이 표면적 |
| 1 | 논거 없음, 주장만 반복 |

---

## R7 — Flow (가독성/독자 경험)

> **평가 질문**: 읽는 도중 방향을 잃지 않는가?  
> **Lv.2 기준**: 문단 단위로 흐름이 유지되는가가 핵심

| 점수 | 기준 |
|------|------|
| 5 | 처음부터 끝까지 자연스럽게 읽히며 각 문단이 다음 문단을 자연스럽게 이끔 |
| 4 | 대체로 잘 읽히나 한 곳에서 독자가 잠깐 멈추게 됨 |
| 3 | 2~3곳에서 "이게 왜 나오지?" 하는 의구심이 생김 |
| 2 | 전반적으로 읽기 불편하고 글의 방향을 자주 잃게 됨 |
| 1 | 읽는 내내 혼란스러워 전체 재독이 필요 |

---

## R8 — Paraphrasing (표현 다양성 / 재진술)

> **Lv.2 적용 범위**: B(반복 회피) + A 시작(자신의 언어로 기초 재진술)  
> **평가 질문**: 같은 개념을 다양하게 표현하고, 주장을 다른 말로 재진술하는가?

| 점수 | 기준 |
|------|------|
| 5 | 같은 개념을 반복 없이 다양하게 표현하고, 핵심 주장이나 논거를 한 번 이상 자신의 말로 재진술 |
| 4 | 대부분 다양하게 표현하나 한 표현이 2~3회 반복되거나 재진술 시도가 어색 |
| 3 | 핵심 표현 반복이 눈에 띄고 어휘 범위가 좁게 느껴짐 |
| 2 | 주요 표현이 문단 전체에 반복되어 단조로움 |
| 1 | 주제 문장이나 핵심 주장을 그대로 복사·반복 |

---

## R9 — Universality (논거·근거의 보편성)

> **Lv.2 기준**: 개인 경험만으로는 2점. 사회적 맥락이나 타인의 경험으로 확장해야 3점  
> **평가 질문**: 논거가 나만의 이야기를 넘어서는가?

| 점수 | 기준 |
|------|------|
| 5 | 논거가 사회적 현상이나 일반적으로 알려진 사실에 기반하며 많은 사람에게 적용됨 |
| 4 | 개인 경험을 기반으로 하되 사회적 맥락이나 타인의 경험으로 확장됨 |
| 3 | 개인 경험 + 간단한 보편화 시도 ("많은 학생들이...", "대부분의 사람들은...") |
| 2 | 개인 경험만 제시, 보편화 시도 없음 |
| 1 | 개인 감정·주관적 판단만, 구체적 사례 없음 |
