# Lv.2 채점 루브릭 — 고등학생

> **핵심 목표**: 논리적 연결 고리를 명시적으로 제시할 수 있는가  
> **합격선**: R1 Gate Pass + 25/50  

---

## R10 — Rhetoric (수사의도)

> **Lv.2 기준**: 주장을 강조하기 위한 기본적인 강조 표현을 사용하는가?

| 점수 | 기준 |
|------|------|
| 5 | "특히", "반드시" 등 강조 어구를 사용해 의도를 선명하게 함 |
| 3 | 의도는 명확하나 강조하려는 지점이 평이함 |
| 1 | 의도가 느껴지지 않는 무미건조한 나열 |

---

## R11 — Diction (언어 통제 / 어휘 선택)

> **Lv.2 기준**: 일상 용어를 넘어 주제와 관련된 어휘를 구사하는가?

| 점수 | 기준 |
|------|------|
| 5 | 주제와 관련된 교과서적 어휘를 정확히 사용하고 문장이 매끄러움 |
| 3 | 어휘 수준은 평이하나 문법적 오류가 적음 |
| 1 | 단어 선택이 부정확하여 문장의 뜻이 왜곡됨 |
