# Lv.3 채점 루브릭 — 수능 논술

> **핵심 목표**: 다각도의 논거를 구성하고 유기적으로 연결할 수 있는가  
> **합격선**: R1 Gate Pass + 30/50  

---

## R10 — Rhetoric (수사의도)

> **Lv.3 기준**: 독자의 주의를 환기하거나 논리적 대비를 보여주는 수사 장치를 쓰는가?

| 점수 | 기준 |
|------|------|
| 5 | "반면", "이와 달리" 등 대비를 명확히 하거나 질문을 통해 독자의 환기를 유도함 |
| 3 | 의도는 명확하나 표현이 상투적임 |
| 1 | 수사적 배려 없이 논리만 나열하여 지루함 |

---

## R11 — Diction (언어 통제 / 어휘 선택)

> **Lv.3 기준**: 단조로운 문장 구조를 탈피하고 유의어를 활용하는가?

| 점수 | 기준 |
|------|------|
| 5 | 같은 단어 반복을 피하며 문장의 길이가 적절히 조절되어 리듬감이 있음 |
| 3 | 어휘가 적절하나 문장 구조가 다소 단조로움 |
| 1 | 어휘 반복이 심해 독해 흐름을 저해함 |
