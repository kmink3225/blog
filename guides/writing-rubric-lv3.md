# Lv.3 채점 루브릭 — 수능 논술

> **핵심 목표**: 5단락 구조를 갖추고 반론을 기초적으로 인식할 수 있는가  
> **합격선**: R1 Gate Pass + 38/50 (75% 고정 — 모든 레벨 공통)  
> **설계 원칙**: 모든 기준의 1~5점은 수능 논술 수준 글의 스펙트럼 안에서 정의된다.  
> Lv.2와의 핵심 차이: Thesis가 서론에 명시되어야 하고, 5단락 구조와 반론 인식이 요구되며, 보편적 근거 1개 이상이 필수다.

---

## R1 — Stance [Gate]

| 판정 | 기준 |
|------|------|
| **Pass** | Thesis가 서론(첫 문단)에 명시되어 있고 한 문장으로 정리됨 |
| **Soft Fail** | Thesis는 있으나 서론이 아닌 중간에 등장하거나 "~라고 생각한다" 수준으로 단정 짓지 못함 |
| **Hard Fail** | Thesis 없음, 또는 주제와 무관 |

---

## R2 — Focus (주제 일관성)

> **평가 질문**: 반론 문단을 포함한 모든 문단이 thesis를 향해 기여하는가?

| 점수 | 기준 |
|------|------|
| 5 | 모든 문단이 thesis에 직접 기여. 반론 문단도 thesis를 강화하는 방식으로 배치됨 |
| 4 | 한 문단이 thesis와 약하게 연결되나 전체 논증 흐름 유지 |
| 3 | 한 문단이 thesis와 느슨하게 연결되어 논증 집중도가 떨어지나 의도는 파악됨 |
| 2 | 두 문단 이상이 주제에서 벗어나거나 thesis와 무관한 내용 포함 |
| 1 | 글의 절반 이상이 thesis와 무관 |

---

## R3 — Syntax (문법/언어 정확성)

> **평가 범위**: 복합 문장 구사 능력, 논술체 어휘 사용, 구어체 배제  
> **Lv.3 기준**: 구어체 표현이 섞이거나 문장 구조가 단조로우면 3점 이하

| 점수 | 기준 |
|------|------|
| 5 | 복합 문장을 자연스럽게 구사하고 논술체 어휘를 정확히 사용. 문법 오류 없음 |
| 4 | 복합 문장을 시도하나 한두 곳 어색함. 오류 1~2개, 의미 전달에 지장 없음 |
| 3 | 문장 구조가 단조롭거나 오류 3~4개, 논술체 어휘가 부족 |
| 2 | 오류가 빈번하거나 구어체 표현이 반복적으로 섞임 |
| 1 | 문법 오류가 의미 전달을 방해하거나 구어체 일색 |

---

## R4 — Cohesion (유기성/연결성)

> **평가 질문**: 문단 간 논리적 연결이 이루어지는가?  
> **Lv.3 기준**: 연결어만으로 전환하는 것은 3점. 앞 문단이 다음 문단을 논리적으로 예고해야 4점 이상

| 점수 | 기준 |
|------|------|
| 5 | 각 문단이 논증의 흐름 속에서 유기적으로 연결됨. 연결어 없이도 흐름이 자연스러움 |
| 4 | 문단 간 연결이 대체로 자연스러우나 한 곳에서 논리적 점프가 있음 |
| 3 | 연결어("다음으로", "반면에")로 전환을 표시하나 문단 간 논리적 연결은 피상적 |
| 2 | 문단 간 연결이 끊기거나 순서가 부자연스러워 독자가 재구성해야 함 |
| 1 | 문단들이 나열 수준, 논리적 흐름 없음 |

---

## R5 — Structure (논리 전개 구조)

> **Lv.3 요구 구조**: 서론(Thesis) → 본론1(논거1) → 본론2(논거2) → 반론 인식 → 결론  
> **평가 질문**: 5단락 구조가 갖춰져 있고 반론을 인식하고 있는가?

| 점수 | 기준 |
|------|------|
| 5 | 서론(Thesis 포함) → 본론1 → 본론2 → 반론 인식·처리 → 결론 5단계 완성, 각 단계가 명확 |
| 4 | 5단락 구조가 있으나 한 단계가 약하거나 간략하게만 처리됨 |
| 3 | 서론-본론-결론 3단락 구조는 있으나 반론 인식 없음 |
| 2 | 서론 또는 결론 중 하나만 있고 본론 구성이 불분명 |
| 1 | 구조 없음, 생각의 흐름대로 나열 |

---

## R6 — Depth (논거 독립성/심화)

> **Lv.3 기준**: 독립적인 논거 2개 본격 요구. 같은 관점의 반복은 4점 이하  
> **평가 질문**: 논거들이 서로 다른 각도에서 thesis를 강화하는가?

| 점수 | 기준 |
|------|------|
| 5 | 독립적인 논거 2개 이상, 각각 다른 관점에서 thesis를 강화 (예: 개인적 측면 + 사회적 측면, 단기 효과 + 장기 효과) |
| 4 | 논거 2개이나 관점이 겹치거나 하나가 다른 하나의 단순 확장에 그침 |
| 3 | 논거 1개이나 충분한 설명과 예시로 뒷받침됨 |
| 2 | 논거 1개이나 설명이 부족하거나 예시와의 연결이 약함 |
| 1 | 논거 없음, 주장만 반복 |

---

## R7 — Flow (가독성/독자 경험)

> **평가 질문**: 서론부터 결론까지 논증이 자연스럽게 전개되는가?  
> **Lv.3 기준**: 5단락 전체의 흐름이 끊기지 않는가가 핵심

| 점수 | 기준 |
|------|------|
| 5 | 서론부터 결론까지 논증이 자연스럽게 전개됨. 독자가 "다음은 뭐지?"를 예측하며 읽힘 |
| 4 | 대체로 잘 읽히나 한 문단에서 독자가 잠깐 멈추게 됨 |
| 3 | 2~3곳에서 논증의 흐름이 끊기거나 의구심이 생김 |
| 2 | 전반적으로 논증 흐름을 추적하기 어렵고 자주 멈추게 됨 |
| 1 | 읽는 내내 논증 구조를 파악할 수 없음 |

---

## R8 — Paraphrasing (표현 다양성 / 재진술)

> **Lv.3 적용 범위**: B(반복 회피) + A(재진술 능숙)  
> **평가 질문**: 논거와 결론을 자신의 언어로 자연스럽게 재진술하는가?

| 점수 | 기준 |
|------|------|
| 5 | 같은 개념을 일관되게 다양한 표현으로 전달하고, 논거나 결론을 자신의 언어로 자연스럽게 재진술 |
| 4 | 대부분 다양하게 표현하나 한두 곳에서 표현 반복 또는 재진술이 어색함 |
| 3 | 핵심 표현 반복이 눈에 띄나 의미 전달에는 지장 없음 |
| 2 | 주요 표현이 반복되어 논술문으로서의 완성도가 떨어짐 |
| 1 | 주장이나 논거를 그대로 복사·반복 |

---

## R9 — Universality (논거·근거의 보편성)

> **Lv.3 기준**: 보편적 근거 1개 이상 필수. 개인 경험+보편화 시도만으로는 3점 이하  
> **평가 질문**: 논거가 사회적·역사적 사례 또는 일반적으로 검증된 현상에 기반하는가?

| 점수 | 기준 |
|------|------|
| 5 | 논거 대부분이 사회적 사례, 역사적 사례, 또는 일반적으로 검증된 현상에 기반함 |
| 4 | 보편적 근거 1개 + 개인·사례 근거 혼합 |
| 3 | 보편적 근거 1개 이상 제시 (사회적 현상, 역사적 사례 등) |
| 2 | 개인 경험 + 보편화 시도만 있고 실제 보편적 근거 없음 |
| 1 | 개인 경험이나 감정만, 보편화 시도도 없음 |

---

## R10 — Rhetoric (수사 효과)

> **Lv.3 적용 범위**: Hook + 대조·병렬·강조가 의도적으로 배치되어 독자를 설득하는지 평가  
> **평가 질문**: 수사 장치가 주장의 무게를 실제로 더해주는가, 아니면 표면 장식에 그치는가?

| 점수 | 기준 |
|------|------|
| 5 | Hook(질문·대조·통계·사례)이 독자 이목을 잡고, 본문에 대조·병렬·강조 중 두 개 이상이 thesis 강화에 전략적으로 기여 |
| 4 | Hook + 수사 장치 1개가 효과적으로 작동하나 한 곳에서 임팩트 약함 |
| 3 | 수사 장치 사용이 있으나 표면적이며 설득력 기여 약함 |
| 2 | 단조로운 진술 중심, 수사 고려 거의 없음 |
| 1 | 평이한 나열, 독자 설득 의도 전혀 없음 |

---

## R11 — Diction (언어 경제성)

> **Lv.3 적용 범위**: 압축 능력 본격 평가. 잉여 수식 제거, 레지스터 일관성 요구  
> **평가 질문**: 긴 풀어쓰기 대신 정확한 한 단어가 반복적으로 선택되는가?

| 점수 | 기준 |
|------|------|
| 5 | 핵심 개념을 정확한 단어로 반복 압축하고 잉여 수식이 없으며 격식체가 일관됨 |
| 4 | 대체로 경제적이나 한두 곳에서 장황한 풀이가 남음 |
| 3 | 일상 어휘와 풀어쓰기가 혼재, 압축 가능한 표현이 곳곳에 보임 |
| 2 | 잉여 수식·장황한 풀이가 지배적, 격식체가 무너짐 |
| 1 | 어휘가 부정확하거나 모호해서 논지를 흐림 |
