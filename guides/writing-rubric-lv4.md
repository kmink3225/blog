# Lv.4 채점 루브릭 — 대학 리포트 / TOEFL

> **핵심 목표**: Evidence 기반 논거를 구성하고 인용 구조를 갖출 수 있는가  
> **합격선**: R1 Gate Pass + 35/50  

---

## R10 — Rhetoric (수사의도)

> **Lv.4 기준**: 독자의 관심을 끌고 논증에 참여시키는 수사적 장치(Hook)가 있는가?

| 점수 | 기준 |
|------|------|
| 5 | 서론에서 Hook(질문, 통계 등)을 효과적으로 사용하고, 의도가 논증 전체를 관통함 |
| 3 | 수사적 장치를 사용하려 노력하나 논증의 무게감과 어울리지 않음 |
| 1 | 수사의도 없이 자료만 요약/나열한 수준 |

---

## R11 — Diction (언어 통제 / 표현 다양성)

> **Lv.4 기준**: 학술적/전문적 어휘를 정확히 사용하고 문장 통제력이 있는가?

| 점수 | 기준 |
|------|------|
| 5 | 학술적 어휘를 적재적소에 배치하며, 복잡한 문장 구조를 세련되게 통제함 |
| 3 | 일상적인 단어 위주로 구성되어 학술적 무게감이 다소 부족함 |
| 1 | 어휘력이 부족하여 논증을 정교하게 표현하지 못함 |
