# Lv.4 채점 루브릭 — 대학 리포트 / TOEFL

> **핵심 목표**: Evidence 기반 논거를 구성하고 인용 구조를 갖출 수 있는가  
> **합격선**: R1 Gate Pass + 28/40  
> **설계 원칙**: 모든 기준의 1~5점은 대학 리포트·TOEFL 수준 글의 스펙트럼 안에서 정의된다.  
> Lv.3과의 핵심 차이: 서론에 논거 방향 예고가 추가되고, 반론을 반박해야 하며, 논거는 구체적 evidence로 뒷받침되어야 한다.

---

## R1 — Stance [Gate]

| 판정 | 기준 |
|------|------|
| **Pass** | Thesis + 앞으로 전개할 논거의 방향이 서론에 함께 제시됨 (예: "X를 주장한다. 왜냐하면 A와 B 때문이다") |
| **Soft Fail** | Thesis는 있으나 논거 방향 예고 없이 주장만 명시됨 (Lv.3 수준에 그침) |
| **Hard Fail** | Thesis 없음, 또는 주제와 무관 |

---

## R2 — Focus (주제 일관성)

> **평가 질문**: Evidence 인용을 포함한 모든 내용이 thesis 강화에 기여하는가?

| 점수 | 기준 |
|------|------|
| 5 | 모든 문단이 thesis에 직접 기여. Evidence도 thesis 강화에만 사용되고 불필요한 내용 없음 |
| 4 | 한 문단이 thesis와 약하게 연결되나 전체 논증 흐름 유지 |
| 3 | 한 문단이 thesis와 느슨하게 연결되어 논증 집중도가 떨어지나 의도는 파악됨 |
| 2 | 두 문단 이상이 주제에서 벗어나거나 evidence가 논점과 무관하게 나열됨 |
| 1 | 글의 절반 이상이 thesis와 무관 |

---

## R3 — Syntax (문법/언어 정확성)

> **평가 범위**: 학술적 어휘 정확성, 복잡한 문장 구조 구사, 격식체 유지  
> **Lv.4 기준**: 일상적 어휘 수준에 머물면 3점 이하. 학술 어휘 오용은 2점 이하

| 점수 | 기준 |
|------|------|
| 5 | 학술적 어휘를 정확히 사용하고 복잡한 문장 구조를 자연스럽게 구사. 문법 오류 없음 |
| 4 | 학술적 어휘 대체로 적절, 문장 구조 다양하나 한두 곳 어색함. 오류 1~2개 |
| 3 | 어휘 선택이 일상적 수준에 머물거나 문장 구조가 단조롭게 반복됨 |
| 2 | 학술 어휘 오용이 반복되거나 오류가 빈번하여 격식체가 무너짐 |
| 1 | 기초 문법 오류가 심해 의미 파악이 어렵거나 구어체 일색 |

---

## R4 — Cohesion (유기성/연결성)

> **평가 질문**: Evidence 인용 후 해석·연결이 자연스럽게 이루어지는가?  
> **Lv.4 기준**: Evidence를 던지고 해석 없이 다음 문단으로 넘어가면 2점 이하

| 점수 | 기준 |
|------|------|
| 5 | 문단 간 전환이 논리의 흐름으로 이루어짐. Evidence 인용 후 해석·연결이 자연스러움 |
| 4 | 문단 간 연결 대체로 자연스러우나 evidence 인용 후 해석 연결이 한 곳 어색함 |
| 3 | 연결어로 전환을 표시하나 evidence와 논점 연결이 피상적 |
| 2 | 문단 간 연결이 끊기거나 evidence가 논점과 연결되지 않은 채 나열됨 |
| 1 | 논리적 흐름 없음, 문단들이 독립적으로 존재 |

---

## R5 — Structure (논리 전개 구조)

> **Lv.4 요구 구조**: 서론(Thesis+논거방향) → 본론1(논거+evidence) → 본론2(논거+evidence) → 반론+반박 → 결론  
> **평가 질문**: 반론을 인식하는 데 그치지 않고 논리적으로 반박하는가?

| 점수 | 기준 |
|------|------|
| 5 | 서론(Thesis+논거방향 예고) → 본론1(논거+evidence) → 본론2(논거+evidence) → 반론+반박 → 결론 완성 |
| 4 | 구조는 완성이나 반론 반박이 약하거나 evidence 연결이 한 곳 부족 |
| 3 | 서론-본론-결론 구조 + 반론 인식은 있으나 반박 없음 (Lv.3 수준에 그침) |
| 2 | 기본 구조는 있으나 evidence 없이 주장만 반복 |
| 1 | 구조 없음, 생각의 흐름대로 나열 |

---

## R6 — Depth (논거 독립성/심화)

> **Lv.4 기준**: 각 논거는 반드시 구체적 evidence로 뒷받침되어야 함  
> **평가 질문**: 논거가 evidence(통계·연구·사례)로 실질적으로 뒷받침되는가?

| 점수 | 기준 |
|------|------|
| 5 | 독립적 논거 2개 이상, 각 논거가 구체적 evidence(통계·연구·사례)로 뒷받침됨 |
| 4 | 논거 2개이나 하나만 evidence로 뒷받침되고 나머지는 일반적 주장 수준 |
| 3 | 논거 1개이나 evidence로 충분히 뒷받침됨 |
| 2 | 논거는 있으나 evidence 없이 일반적 주장에 그침 |
| 1 | 논거 없음, 주장만 반복 |

---

## R7 — Flow (가독성/독자 경험)

> **평가 질문**: Evidence 인용이 논증 흐름을 방해하지 않고 자연스럽게 녹아드는가?

| 점수 | 기준 |
|------|------|
| 5 | 서론-본론-반론-결론 전체가 하나의 논증으로 읽힘. Evidence 인용이 흐름을 방해하지 않음 |
| 4 | 대체로 잘 읽히나 evidence 인용 부분에서 한 번 흐름이 끊김 |
| 3 | 2~3곳에서 논증 흐름이 끊기거나 evidence가 갑자기 등장하여 맥락이 끊김 |
| 2 | 전반적으로 논증 흐름을 추적하기 어렵고 evidence와 논점 간 연결을 독자가 스스로 구성해야 함 |
| 1 | 읽는 내내 논증 구조 파악 불가 |

---

## R8 — Paraphrasing (표현 다양성 / 재진술)

> **Lv.4 적용 범위**: B(반복 회피) + A(재진술 능숙) + C 시작(외부 내용 재진술)  
> **평가 질문**: 인용한 evidence나 외부 주장을 자신의 언어로 정확히 재진술하는가?

| 점수 | 기준 |
|------|------|
| 5 | 반복 없이 다양하게 표현하고, 외부 주장·evidence를 자신의 언어로 정확하고 자연스럽게 재진술 |
| 4 | 대부분 다양하게 표현하나 외부 내용 재진술 시 원문 표현에 가깝게 의존하는 곳이 있음 |
| 3 | 핵심 표현 반복이 눈에 띄거나 외부 내용을 거의 그대로 옮김 |
| 2 | 주요 표현 반복 + 외부 내용을 직접 인용하는 방식에만 의존 |
| 1 | 주장·논거를 그대로 복사·반복 |

---

## R9 — Universality (논거·근거의 보편성)

> **Lv.4 기준**: 구체적 데이터·연구 결과·통계가 요구됨. 사회·역사적 사례만으로는 3점  
> **평가 질문**: 논거가 검증 가능한 근거(데이터·연구)에 기반하는가?

| 점수 | 기준 |
|------|------|
| 5 | 논거 대부분이 구체적 데이터·연구 결과·통계에 기반. 출처가 명시되거나 암시됨 |
| 4 | 검증된 근거 1개 + 사회적·역사적 사례 혼합 |
| 3 | 사회적·역사적 사례 중심, 데이터·연구 언급 없음 (Lv.3 수준에 그침) |
| 2 | 개인 경험 + 보편화 시도만, 사회적 근거가 약함 |
| 1 | 개인 감정·주관적 판단만, 보편화 시도 없음 |
