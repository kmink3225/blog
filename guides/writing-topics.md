# Writing 주제 선정 서브 스킬

> `random` 트리거 감지 시 `writing-tutor.md`가 이 파일을 로드한다.
> 아래 알고리즘을 따라 주제를 선정하고, 결과를 writing-tutor로 반환한다.

---

## 주제 선정 알고리즘

```
Step 1: 그룹 번호 1~15 중 uniform random 선택 → 그룹 확정
Step 2: 레벨 → 티어 매핑 (아래 표 참조)
Step 3: 해당 그룹의 해당 티어 후보 3개 중 uniform random 선택 → 주제 확정
Step 4: 선정 결과 출력 형식:
  "[그룹 N: {그룹명}] {주제}"
  → writing-tutor.md로 반환, 즉시 출제 진행
```

**레벨 → 티어 매핑**

| 레벨 | 티어 | 주제 수준 |
|------|------|----------|
| Lv.1~2 | A | 중학생~고등학생 — 일상적 사회 현안, 도덕·윤리 판단, 전문 지식 불필요 |
| Lv.3~4 | B | 수능~대학 리포트 — 사회·제도적 주제, 일반 상식으로 논증 가능 |
| Lv.5~6 | C | GRE~실무 보고서 — 구조적·정책적 사고, 분야 개념 등장 |
| Lv.7   | D | 학술 논문 — Contribution 필요, 방법론 논의 가능 |
