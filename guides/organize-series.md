# 시리즈 블로그 정리

블로그 시리즈를 체계적으로 정리할 때 따르는 패턴.

> **선행 조건**: 이 스킬을 실행하기 전에 `guides/info-search.md`의 병렬 탐색 프로토콜을 먼저 실행한다.

## 절차

### Step 1: 기존 파일 전체 읽기

현재 상태를 파악한다.

### Step 2: 문제 진단

중복, 잘못된 내용, 누락된 주제, 비체계적 구성을 식별한다.

### Step 3: 구조 설계

기초 → 핵심 → 심화 → 실전 순서로 배치한다.

### Step 4: 파일 처리

- 잘 된 파일은 유지
- 잘못된 파일은 재작성
- 누락된 주제는 추가

### Step 5: 요약 테이블 제공

작업 결과를 사용자에게 보고한다.
