# Lv.5 채점 루브릭 — GRE AW

> **핵심 목표**: Toulmin 논증 구조를 구사하고 반론을 자기 주장 강화에 통합할 수 있는가  
> **합격선**: R1 Gate Pass + 40/50  
> **설계 원칙**: 모든 기준의 1~5점은 GRE AW 수준 글의 스펙트럼 안에서 정의된다.  
> Lv.4와의 핵심 차이: 주장에 조건(Qualifier)을 명시하고, 논거에 연결 논리(Warrant)를 갖추며, Evidence의 한계를 스스로 인식한다.

---

## Toulmin 구조 참조

| 요소 | 설명 | 예시 |
|------|------|------|
| Claim | 주장 | "원격근무는 생산성을 높인다" |
| Grounds | 논거·evidence | "Stanford 연구: 재택근무자 생산성 13% 향상" |
| Warrant | 연결 논리 | "통제된 환경에서의 집중력이 생산성의 핵심 변수이기 때문" |
| Qualifier | 조건 한정 | "자기 통제력이 있는 직군에 한해서" |
| Rebuttal | 반론 통합 | "협업 의존도가 높은 직군에는 적용되지 않으나, 이는 직군별 유연 적용으로 해결 가능" |

---

## R1 — Stance [Gate]

| 판정 | 기준 |
|------|------|
| **Pass** | Thesis + 논거 방향 예고 + 주장의 적용 조건·범위 한정(Qualifier)이 서론에 함께 제시됨 |
| **Soft Fail** | Thesis + 논거 방향은 있으나 Qualifier 없이 조건 없는 절대적 주장만 함 (Lv.4 수준에 그침) |
| **Hard Fail** | Thesis 없음, 또는 주제와 무관 |

---

## R2 — Focus (주제 일관성)

> **평가 질문**: Qualifier와 Rebuttal을 포함한 모든 요소가 thesis를 정교화하는 방향으로 기여하는가?

| 점수 | 기준 |
|------|------|
| 5 | 모든 문단이 thesis에 직접 기여. Qualifier와 Rebuttal도 thesis를 약화시키지 않고 정교화함 |
| 4 | 한 문단이 thesis와 약하게 연결되나 전체 논증 흐름 유지 |
| 3 | 한 문단이 thesis와 느슨하게 연결되어 논증 집중도가 떨어지나 의도는 파악됨 |
| 2 | 두 문단 이상이 주제에서 벗어나거나 Rebuttal이 오히려 자기 주장을 약화시킴 |
| 1 | 글의 절반 이상이 thesis와 무관 |

---

## R3 — Syntax (문법/언어 정확성)

> **평가 범위**: GRE 수준의 정교한 학술 어휘, 뉘앙스 있는 표현, 복잡한 문장 구조  
> **Lv.5 기준**: 대학 리포트 수준의 어휘로 그치면 3점 이하. 어휘의 뉘앙스 오용은 2점 이하

| 점수 | 기준 |
|------|------|
| 5 | GRE 수준의 어휘를 뉘앙스까지 정확하게 사용. 복잡한 문장 구조가 의미를 오히려 명확히 함. 오류 없음 |
| 4 | 고급 어휘 대체로 적절, 문장 구조 다양하나 한두 곳 어색하거나 어휘 선택이 아쉬움 |
| 3 | 어휘가 대학 리포트 수준에 머물거나 문장 구조가 충분히 정교하지 않음 |
| 2 | 고급 어휘 오용이 반복되거나 단순 문장 구조 일색 |
| 1 | 기초 문법 오류로 의미 파악이 어렵거나 어휘·표현이 Lv.2 이하 수준 |

---

## R4 — Cohesion (유기성/연결성)

> **평가 질문**: Claim→Grounds→Warrant 연결이 논리적으로 이루어지는가?  
> **Lv.5 기준**: Warrant(연결 논리) 없이 Claim→Evidence를 직결하면 3점 이하

| 점수 | 기준 |
|------|------|
| 5 | Claim→Grounds→Warrant 연결이 논리적으로 이루어짐. 연결어 없이도 논리 흐름이 자연스럽게 형성됨 |
| 4 | 전체 흐름 자연스러우나 Warrant 설명이 한 곳 부족하거나 암묵적 |
| 3 | 연결어로 전환을 표시하나 Warrant 없이 Claim→Evidence가 직결됨 |
| 2 | 논증 단계 간 연결이 끊기거나 Rebuttal이 논증 흐름을 방해 |
| 1 | 논리적 흐름 없음, 문단들이 독립적으로 존재 |

---

## R5 — Structure (논리 전개 구조)

> **Lv.5 요구 구조**: Toulmin 전체 — Claim → Grounds → Warrant → Qualifier → Rebuttal(자기 주장 강화)  
> **평가 질문**: 반론이 자기 주장을 약화시키지 않고 오히려 강화하는 방식으로 통합되는가?

| 점수 | 기준 |
|------|------|
| 5 | Toulmin 전체(Claim+Grounds+Warrant+Qualifier+Rebuttal) 완성. 반론이 자기 주장을 강화하는 방식으로 통합됨 |
| 4 | Toulmin 구조 대부분 갖춰지나 Warrant 또는 Qualifier 중 하나가 약하거나 생략됨 |
| 3 | Claim+Grounds+Rebuttal은 있으나 Warrant가 명시적이지 않음 (Lv.4 수준에 그침) |
| 2 | 기본 구조 + 반론 반박은 있으나 Toulmin 구조로의 통합 없음 |
| 1 | 구조 없음, 생각의 흐름대로 나열 |

---

## R6 — Depth (논거 독립성/심화)

> **Lv.5 기준**: Warrant(연결 논리)가 명시되어야 하고, Evidence의 한계·조건을 스스로 인식해야 함  
> **평가 질문**: "이 evidence가 왜 이 claim을 뒷받침하는가?"를 논리적으로 설명하는가?

| 점수 | 기준 |
|------|------|
| 5 | 독립적 논거 2개 이상 + 각 논거의 Warrant 명시 + evidence의 한계·적용 조건까지 인식 |
| 4 | 독립적 논거 2개 + Warrant 있으나 evidence 한계 인식 없음 |
| 3 | 독립적 논거 2개 + evidence 뒷받침, Warrant는 암묵적 (Lv.4 수준) |
| 2 | 논거 있으나 Warrant 없이 Claim→Evidence 직결 |
| 1 | 논거 없음, 주장만 반복 |

---

## R7 — Flow (가독성/독자 경험)

> **평가 질문**: Qualifier와 Rebuttal이 논증을 강화하는 방식으로 자연스럽게 배치되는가?

| 점수 | 기준 |
|------|------|
| 5 | 논증 전체가 하나의 정교한 흐름. Qualifier와 Rebuttal이 논증을 강화하는 방식으로 자연스럽게 배치됨 |
| 4 | 대체로 정교하게 읽히나 Qualifier 또는 Rebuttal 부분에서 한 번 흐름이 끊김 |
| 3 | 2~3곳에서 논증 흐름이 끊기거나 Rebuttal이 갑자기 등장하여 맥락이 끊김 |
| 2 | 전반적으로 논증 흐름을 추적하기 어렵고 Toulmin 각 요소 간 연결을 독자가 스스로 구성해야 함 |
| 1 | 읽는 내내 논증 구조 파악 불가 |

---

## R8 — Paraphrasing (표현 다양성 / 재진술)

> **Lv.5 적용 범위**: B+A+C 본격 적용 — 외부 주장·evidence를 정확하게 요약·변환하여 자신의 논증에 통합  
> **평가 질문**: 인용한 연구·주장을 원문 없이도 자신의 논증 맥락에 맞게 재구성하는가?

| 점수 | 기준 |
|------|------|
| 5 | 외부 주장·evidence를 정확하게 요약·변환하여 자신의 논증에 자연스럽게 통합. 반복 없음 |
| 4 | 외부 내용 요약·변환을 시도하나 원문 표현에 일부 의존하거나 어색하게 통합됨 |
| 3 | 외부 내용 재진술은 있으나 요약·변환보다 거의 그대로 옮기는 경향 |
| 2 | 외부 내용을 직접 인용에만 의존하거나 핵심 표현 반복 |
| 1 | 주장·논거를 그대로 복사·반복 |

---

## R9 — Universality (논거·근거의 보편성)

> **Lv.5 기준**: Evidence의 방법론적 신뢰성과 적용 범위 한계까지 인식해야 5점  
> **평가 질문**: 인용한 연구·데이터가 어떤 조건에서 유효한지 스스로 인식하는가?

| 점수 | 기준 |
|------|------|
| 5 | 논거가 방법론적으로 신뢰할 수 있는 연구·데이터에 기반하고, 적용 범위와 한계를 명시적으로 인식 |
| 4 | 구체적 데이터·연구 기반 논거 + 적용 범위 인식 있으나 방법론 신뢰성 언급 없음 |
| 3 | 구체적 데이터·연구 기반 논거, 한계·범위 인식 없음 (Lv.4 수준에 그침) |
| 2 | 사회·역사적 사례 중심, 데이터·연구 근거 없음 |
| 1 | 개인 경험·감정만, 보편화 시도 없음 |

---

## R10 — Rhetoric (수사 효과)

> **Lv.5 적용 범위**: Pathos/logos/ethos 배합 + tone modulation 본격 평가  
> **평가 질문**: 수사 장치가 논증 강화, 신뢰 구축, 독자 납득을 균형 있게 달성하는가?

| 점수 | 기준 |
|------|------|
| 5 | Hook, 대조·병렬·강조, tone modulation이 pathos/logos/ethos 배합을 이루어 독자를 논증 안으로 끌어들임 |
| 4 | 주요 수사 장치가 효과적이나 한 축(예: ethos 구축)이 약하거나 톤 변화가 한 곳에서 어색함 |
| 3 | 수사 장치가 있으나 단일 축에만 의존(예: 논리만 강조)하여 설득력 한정적 |
| 2 | 수사 장치 사용이 표면적이고 임팩트 미약 |
| 1 | 단조로운 진술, 수사 전략 전무 |

---

## R11 — Diction (언어 경제성)

> **Lv.5 적용 범위**: 고급 어휘를 경제적·정확하게 구사하는지 평가. 장황한 풀이는 3점 이하  
> **평가 질문**: 긴 개념 설명을 정확한 한 단어로 대체하는 사례가 반복되는가? 레지스터가 학술 톤에 일관된가?

| 점수 | 기준 |
|------|------|
| 5 | 고급 어휘를 정확·경제적으로 구사하고 긴 풀이를 한 단어로 압축하는 사례 다수. 레지스터 일관 |
| 4 | 대부분 경제적이나 한두 곳에서 장황한 풀이가 남거나 레지스터가 한 번 흔들림 |
| 3 | 고급 어휘 사용 범위는 있으나 압축이 부족하고 풀어쓰기가 자주 보임 |
| 2 | 잉여 수식·장황한 풀이가 반복되어 논증이 느슨해짐 |
| 1 | 어휘 선택 부정확, 레지스터 불일치로 학술적 엄밀성 훼손 |
