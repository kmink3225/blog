# Lv.7 채점 루브릭 — 학술 논문

> **핵심 목표**: Peer-review를 통과할 수 있는 수준으로 방법론을 방어하고 학술적 기여를 명시할 수 있는가  
> **합격선**: R1 Gate Pass + 38/50 (75% 고정 — 모든 레벨 공통)  
> **설계 원칙**: 모든 기준의 1~5점은 학술 논문 수준 글의 스펙트럼 안에서 정의된다.  
> Lv.6과의 핵심 차이: 의사결정 지원이 아니라 학문적 기여가 목적이다. 선행 연구와의 관계를 정립하고, 방법론을 방어하며, 결과의 일반화 가능성과 한계를 학술적으로 논의해야 한다.

---

## 학술 논문 구조 참조

| 섹션 | 역할 |
|------|------|
| Introduction | 연구 배경 → 선행 연구 gap → 본 연구의 contribution + Thesis |
| (Literature Review) | 선행 연구 체계적 정리 — 긴 논문에 해당 |
| Methodology | 연구 방법 설계·선택 근거 방어 |
| Results | 데이터·분석 결과 제시 |
| Discussion | 결과 해석 + 선행 연구와 비교 + 함의 |
| Conclusion | 기여 요약 + 한계 + 후속 연구 제언 |

---

## R1 — Stance [Gate]

| 판정 | 기준 |
|------|------|
| **Pass** | Thesis + 학술적 기여(contribution) + 선행 연구와의 차별점이 서론에 명시됨 |
| **Soft Fail** | Thesis + contribution은 있으나 선행 연구와의 관계 정립 없음 (Lv.6 수준에 그침) |
| **Hard Fail** | Thesis 없음, contribution 없음, 또는 연구 목적 불명확 |

---

## R2 — Focus (주제 일관성)

> **평가 질문**: 선행 연구 리뷰를 포함한 모든 섹션이 contribution을 향해 기여하는가?

| 점수 | 기준 |
|------|------|
| 5 | 모든 섹션이 contribution에 직접 기여. 선행 연구 리뷰도 gap 설정에 직접 연결됨 |
| 4 | 한 섹션이 contribution과 약하게 연결되나 전체 논증 방향 유지 |
| 3 | 선행 연구 리뷰가 과도하거나 contribution과의 연결이 느슨하여 집중도가 떨어짐 |
| 2 | 두 섹션 이상이 contribution과 무관하거나 연구 목적과 방향이 엇갈림 |
| 1 | 절반 이상이 contribution과 무관 |

---

## R3 — Syntax (문법/언어 정확성)

> **평가 범위**: 분야별 학술 전문 용어 정확 사용, academic writing 규범 준수, 모호한 표현 완전 배제  
> **Lv.7 기준**: 전문 용어 오용은 2점 이하. 실무 보고서체 잔재는 3점 이하

| 점수 | 기준 |
|------|------|
| 5 | 분야별 학술 전문 용어를 뉘앙스까지 정확히 사용. Academic writing 규범 완벽 준수. 모호한 표현 없음 |
| 4 | 전문 용어 대체로 정확, 한두 곳 표현이 불명확하거나 규범에서 벗어남 |
| 3 | 전문 용어 사용이 부정확하거나 academic writing 규범이 불완전하게 적용됨 |
| 2 | 전문 용어 오용이 반복되거나 표현이 모호하여 학술적 엄밀성이 떨어짐 |
| 1 | 기초적 오류로 의미 파악이 어렵거나 완전한 비학술적 표현 |

---

## R4 — Cohesion (유기성/연결성)

> **평가 질문**: Literature gap → Methodology → Results → Discussion이 하나의 논증으로 유기적으로 연결되는가?  
> **Lv.7 기준**: Results와 Discussion의 연결, 또는 gap과 Methodology의 연결이 피상적이면 3점 이하

| 점수 | 기준 |
|------|------|
| 5 | Literature gap → Methodology → Results → Discussion → Conclusion이 하나의 논증으로 유기적으로 연결됨 |
| 4 | 전체 흐름 자연스러우나 한 섹션에서 연결 논리가 약하거나 암묵적 |
| 3 | 섹션 간 전환은 있으나 gap-Methodology 연결 또는 Results-Discussion 연결이 피상적 |
| 2 | 섹션들이 독립적으로 존재하거나 논증의 흐름이 단절됨 |
| 1 | 논리적 흐름 없음 |

---

## R5 — Structure (논리 전개 구조)

> **Lv.7 요구 구조**: Introduction(gap+contribution) → Methodology(방어) → Results → Discussion(선행 연구 비교) → Conclusion(기여+한계+후속)  
> **평가 질문**: Reviewer가 방법론의 타당성에 의문을 제기했을 때 글 자체로 방어 가능한가?

| 점수 | 기준 |
|------|------|
| 5 | Introduction(gap+contribution) → Methodology(방법론 방어) → Results → Discussion(선행 연구 비교·해석) → Conclusion(기여+한계+후속 연구) 완성 |
| 4 | 전체 구조 있으나 방법론 방어 또는 Discussion의 선행 연구 연결 중 하나가 약함 |
| 3 | 기본 구조(서론-방법-결과-결론)는 있으나 방법론 방어와 선행 연구 비교 없음 (Lv.6 수준에 그침) |
| 2 | 구조의 흔적은 있으나 학술 논문 형식 미적용 |
| 1 | 구조 없음, 생각의 흐름대로 나열 |

---

## R6 — Depth (논거 독립성/심화)

> **Lv.7 기준**: 각 논거는 방법론적 근거를 갖춰야 하고, 선행 연구와의 차별점이 명확해야 함  
> **평가 질문**: "왜 이 방법으로 이 결론에 도달했는가?"를 학술적으로 방어하는가?

| 점수 | 기준 |
|------|------|
| 5 | 독립적 논거 2개 이상 + 각 논거의 방법론적 근거 명시 + 선행 연구와의 차별점·기여 명확 |
| 4 | 독립적 논거 2개 + 방법론적 근거 있으나 선행 연구와의 차별점 불명확 |
| 3 | 독립적 논거 2개 + evidence 뒷받침, 방법론 방어 없음 (Lv.5 수준에 그침) |
| 2 | 논거 있으나 방법론적 근거 없이 주장 수준에 그침 |
| 1 | 논거 없음, 주장만 반복 |

---

## R7 — Flow (가독성/독자 경험)

> **평가 질문**: Reviewer(비판적 독자)가 각 섹션의 역할과 필요성을 자연스럽게 납득하는가?

| 점수 | 기준 |
|------|------|
| 5 | Introduction의 gap → Methodology → Results → Discussion이 하나의 일관된 학술 논증으로 읽힘. Reviewer가 각 단계의 필요성을 자연스럽게 납득 |
| 4 | 대체로 잘 읽히나 한 섹션에서 논증 흐름이 끊기거나 섹션의 역할이 불명확 |
| 3 | 2~3곳에서 논증 흐름이 끊기거나 섹션 간 논리적 연결을 독자가 스스로 구성해야 함 |
| 2 | 전반적으로 논증 흐름 추적이 어렵고 각 섹션의 역할이 불명확 |
| 1 | 읽는 내내 논증 구조 파악 불가 |

---

## R8 — Paraphrasing (표현 다양성 / 재진술)

> **Lv.7 적용 범위**: B+A+C 완숙 + 선행 연구 정확 요약·인용 + 자신의 연구와의 연결  
> **평가 질문**: 선행 연구를 원문 없이 자신의 논증 맥락에 맞게 정확히 재구성하여 gap과 contribution에 연결하는가?

| 점수 | 기준 |
|------|------|
| 5 | 선행 연구를 정확히 요약·변환하여 자신의 연구의 gap과 contribution에 자연스럽게 연결. 원문 의존 없음 |
| 4 | 선행 연구 요약·변환은 하나 연결 논리가 한 곳 약하거나 원문 표현에 일부 의존 |
| 3 | 선행 연구 재진술은 있으나 자신의 연구와의 연결 없이 나열에 그침 |
| 2 | 선행 연구를 직접 인용에만 의존하거나 핵심 표현 반복 |
| 1 | 주장·논거를 그대로 복사·반복 |

---

## R9 — Universality (논거·근거의 보편성)

> **Lv.7 기준**: 연구 결과의 일반화 가능성(generalizability)과 외적 타당도를 학술적으로 논의해야 함  
> **평가 질문**: 이 연구 결과가 어떤 맥락에서 일반화될 수 있고, 어떤 조건에서 한계가 있는가?

| 점수 | 기준 |
|------|------|
| 5 | 연구 결과의 일반화 가능성과 외적 타당도를 명시적으로 논의. 적용 가능 맥락과 한계를 학술적으로 제시 |
| 4 | 일반화 가능성 언급 있으나 외적 타당도 논의 부족하거나 한계 인식이 표면적 |
| 3 | 연구 결과 보고에 그치고 일반화 가능성 논의 없음 (Lv.6 수준에 그침) |
| 2 | 특정 사례·맥락에만 적용되는 주장, 일반화 시도 없음 |
| 1 | 개인 경험·감정 수준, 학술적 근거 없음 |

---

## R10 — Rhetoric (수사 효과)

> **Lv.7 적용 범위**: 학술 톤을 유지하면서 gap·contribution의 의의를 reviewer에게 설득하는 수사 평가  
> **평가 질문**: 수사 장치가 학술적 엄밀성을 해치지 않고 gap의 중요성과 contribution의 의의를 강조하는가?

| 점수 | 기준 |
|------|------|
| 5 | Gap의 중요성·contribution의 의의를 강조하는 수사 장치가 학술 톤과 조화롭게 결합되어 reviewer를 설득 |
| 4 | 수사 장치가 효과적이나 한 곳에서 학술 톤이 흔들리거나 강조가 과함 |
| 3 | 수사 고려가 간헐적이고 gap·contribution의 임팩트가 약하게 전달됨 |
| 2 | 단조로운 학술 진술, 의의 강조 장치 없음 / 또는 과장된 표현으로 학술 톤 훼손 |
| 1 | 수사 전략 부재 또는 과장으로 논문의 신뢰도 훼손 |

---

## R11 — Diction (언어 경제성)

> **Lv.7 적용 범위**: 학술 전문 용어의 정밀한 뉘앙스 + 완전한 잉여 제거 요구  
> **평가 질문**: 모든 단어가 논증에 필수적이며, 한 단어로 압축 가능한 설명이 남아 있지 않은가?

| 점수 | 기준 |
|------|------|
| 5 | 전문 용어의 뉘앙스를 정확하게 구사하고 모든 단어가 논증에 필수적. 잉여 수식·장황한 풀이 완전 부재 |
| 4 | 대부분 정밀·경제적이나 한두 곳에서 장황한 풀이 또는 부정확한 뉘앙스 선택 |
| 3 | 전문 용어는 정확하나 잉여 수식·풀어쓰기가 남아 학술적 밀도가 떨어짐 |
| 2 | 전문 용어 뉘앙스 선택이 부정확하거나 장황함이 반복되어 논증 밀도 훼손 |
| 1 | 전문 용어 오용 또는 구어체 잔재로 학술적 엄밀성 상실 |
