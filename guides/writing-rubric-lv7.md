# Lv.7 채점 루브릭 — 학술 논문 / 최고위직 연설문

> **핵심 목표**: 분야를 선도하는 통찰을 압도적인 품격으로 전달하는가  
> **합격선**: R1 Gate Pass + 46/50  

---

## R10 — Rhetoric (수사의도)

> **Lv.7 기준**: 글 전체의 톤(Tone)과 스타일로 독자를 압도하고 신뢰를 확보하는가?

| 점수 | 기준 |
|------|------|
| 5 | 글 전체가 하나의 예술적/학술적 완성도를 지니며, 수사가 논리의 권위를 높임 |
| 3 | 논리는 완벽하나 품격이나 스타일 면에서 최고 수준에 도달하지 못함 |
| 1 | 수사적 품격이 없어 주제의 중요성을 떨어뜨림 |

---

## R11 — Diction (언어 통제 / 독보적 완성도)

> **Lv.7 기준**: 단어 하나가 복잡한 개념을 함축하며 완벽한 문장 리듬을 보이는가?

| 점수 | 기준 |
|------|------|
| 5 | 단어 선택이 독보적이며 문장의 리듬과 호흡이 완벽하게 통제됨 |
| 3 | 어휘가 매우 정교하나 문장 리듬이 다소 정형적임 |
| 1 | 어휘 오용이나 문장 통제력 상실로 권위가 실추됨 |
