# Lv.1 채점 루브릭 — 중학생

> **핵심 목표**: 주장과 근거를 구분할 수 있는가  
> **합격선**: R1 Gate Pass + 38/50 (75% 고정 — 모든 레벨 공통)  
> **설계 원칙**: 모든 기준의 1~5점은 중학생 글의 스펙트럼 안에서 정의된다.  
> 5/5를 받으려면 중학생으로서 탁월해야 하고, 1/5는 중학생 수준에서도 최하이다.

---

## R1 — Stance [Gate]

| 판정 | 기준 |
|------|------|
| **Pass** | 찬성 또는 반대 중 한쪽 입장이 글 어딘가에 보임 |
| **Soft Fail** | 입장이 있는 것 같으나 글 중간에 반대 의견도 자기 생각처럼 쓰여 있음 |
| **Hard Fail** | 입장이 없거나, 주제와 전혀 관계없는 내용만 있음 |

---

## R2 — Focus (주제 일관성)

> **평가 질문**: 글 전체가 같은 주제를 향하고 있는가?

| 점수 | 기준 |
|------|------|
| 5 | 처음부터 끝까지 주제에서 벗어나는 문장이 단 하나도 없음 |
| 4 | 마지막 문단에서 한두 문장이 살짝 다른 이야기로 빠지나 주제 의도는 명확 |
| 3 | 중간에 관련 없는 개인 에피소드가 한 문단 정도 끼어 있으나 전체 의도는 파악됨 |
| 2 | 관련 없는 이야기가 글의 절반 가까이 차지하여 주제가 흐릿해짐 |
| 1 | 글을 읽어도 무엇을 주장하려는지 주제 자체가 파악되지 않음 |

---

## R3 — Syntax (문법/언어 정확성)

> **평가 범위**: 맞춤법·띄어쓰기·기본 문장 끝맺음 (어휘 수준·문체는 평가하지 않음)

| 점수 | 기준 |
|------|------|
| 5 | 맞춤법·띄어쓰기 오류가 없고 모든 문장이 자연스럽게 끝맺음 |
| 4 | 오류 1~2개, 읽는 데 지장 없음 |
| 3 | 오류 3~5개, 주의하면 의미 파악 가능 |
| 2 | 오류가 6개 이상 또는 한 문단에 집중되어 읽다가 멈추게 됨 |
| 1 | 문장 단위로 의미를 알아보기 어려울 정도의 오류가 전반에 퍼져 있음 |

---

## R4 — Cohesion (유기성/연결성)

> **평가 질문**: 문장들이 순서대로 자연스럽게 이어지는가?  
> **Lv.1 기준**: "왜냐하면", "그래서", "예를 들어" 같은 연결어의 적절한 사용이 핵심

| 점수 | 기준 |
|------|------|
| 5 | 연결어("왜냐하면", "그러므로", "예를 들어" 등)를 적절히 사용하여 문장이 자연스럽게 이어짐 |
| 4 | 연결어를 사용하나 한 곳에서 어색한 전환이 있음 |
| 3 | 연결어가 있지만 "그리고", "또" 수준의 단순 나열에 그침 |
| 2 | 연결어가 거의 없어 문장들이 각각 단독으로 존재하는 느낌 |
| 1 | 문장 간 연결이 전혀 없어 독자가 순서를 스스로 재구성해야 함 |

---

## R5 — Structure (논리 전개 구조)

> **Lv.1 요구 구조**: Claim(주장) + Example(근거/예시) + 마무리 3단계  
> **평가 질문**: 주장과 근거가 구분되어 있고 마무리 문장으로 정리되는가?

| 점수 | 기준 |
|------|------|
| 5 | 주장이 명확하고 → 근거/예시가 주장을 직접 뒷받침하며 → 마무리 문장이 있음 |
| 4 | 주장과 근거가 있으나 마무리가 없거나, 주장-근거 연결이 한 곳에서 어색함 |
| 3 | 주장 또는 근거 중 하나는 있으나 둘 다 명확하지는 않음 |
| 2 | 주장도 근거도 불분명하나 무언가 말하려는 시도는 보임 |
| 1 | 구조 없음, 느낌이나 감상만 나열됨 |

---

## R6 — Depth (논거 독립성/심화)

> **Lv.1 기준**: 근거 1개라도 주장과 연결되면 3점(기본 충족)  
> **평가 질문**: 근거가 얼마나 주장을 뒷받침하는가?

| 점수 | 기준 |
|------|------|
| 5 | 서로 다른 2개의 근거가 각각 독립적으로 주장을 뒷받침함 (예: 본인 경험 + 주변 관찰) |
| 4 | 근거 2개이나 사실상 같은 이야기를 반복한 수준 |
| 3 | 근거 1개, 주장과 명확하게 연결됨 |
| 2 | 근거가 있으나 "나는 그냥 그게 싫다" 수준으로 주장과의 연결이 불분명 |
| 1 | 근거 없음, 주장만 반복 |

---

## R7 — Flow (가독성/독자 경험)

> **평가 질문**: 읽다가 막히지 않는가?  
> **Lv.1 기준**: 전문적 흐름이 아니라 "중학생 글로서 읽을 만한가"가 기준

| 점수 | 기준 |
|------|------|
| 5 | 처음부터 끝까지 막힘 없이 읽힘. 중학생 글이지만 흐름이 자연스러움 |
| 4 | 한 문장에서 "이게 왜 여기 있지?" 하는 느낌이 한 번 있음 |
| 3 | 2~3곳에서 잠깐 멈칫하나 전체적으로 읽어나갈 수 있음 |
| 2 | 전반적으로 읽기 불편하고 여러 번 멈추게 됨 |
| 1 | 처음부터 끝까지 읽는 것 자체가 힘듦 |

---

## R8 — Paraphrasing (표현 다양성)

> **Lv.1 적용 범위**: 동일 표현 반복 회피만 평가 (재진술·인용 변환은 평가하지 않음)  
> **평가 질문**: 같은 개념을 쓸 때 다른 표현을 사용하는가?

| 점수 | 기준 |
|------|------|
| 5 | 같은 개념을 반복할 때 다른 표현으로 바꿔 씀 (예: "신상 공개" → "범죄자 정보 공개" → "이름과 얼굴을 알리는 것") |
| 4 | 대부분 다른 표현을 쓰지만 한 표현을 2~3회 그대로 반복함 |
| 3 | 핵심 단어·표현을 4회 이상 반복, 어휘 범위가 좁게 느껴짐 |
| 2 | 주제 문장의 핵심 표현이 거의 모든 문장에 그대로 등장 |
| 1 | 주제 문장을 그대로 복사·반복하거나 단어 순서만 바꾸어 재사용 |

---

## R9 — Universality (논거·근거의 보편성)

> **Lv.1 적용 범위**: 개인 사례 허용. 보편성을 향한 시도가 있는가만 확인  
> **평가 질문**: 내 경험이 나만의 이야기인가, 아니면 더 넓은 이야기로 연결하려 했는가?

| 점수 | 기준 |
|------|------|
| 5 | 개인 경험을 근거로 쓰면서 "이런 일이 나에게만이 아니라 많은 학생/사람들에게도 일어날 수 있다"는 보편화가 명시적으로 있음 |
| 4 | 개인 경험을 쓰면서 "친구들도 이런 경험이 있을 것이다" 정도의 간단한 보편화 표현이 한 문장이라도 있음 |
| 3 | 개인 경험만 제시, 보편화 시도는 없음 (Lv.1 평균 수준) |
| 2 | 경험도 없고 "나는 그냥 싫다/좋다" 수준의 감정 표현만 있음 |
| 1 | 논거 자체 없음, 주장만 있거나 주제 문장 반복 |

---

## R10 — Rhetoric (수사 효과)

> **Lv.1 적용 범위**: 서두가 독자의 흥미를 유발하는가만 평가. 본격 수사 장치(대조·병렬)는 평가하지 않음  
> **평가 질문**: 첫 문장을 읽고 독자가 "계속 읽고 싶다"라는 생각이 드는가?

| 점수 | 기준 |
|------|------|
| 5 | 질문, 친숙한 사례, 또는 인상적인 표현으로 첫 문장이 독자의 관심을 끌고 주장으로 자연스럽게 이어짐 |
| 4 | 첫 문장이 무난하나 본문 중 한 문장 정도가 강조 의도로 쓰여 있음 |
| 3 | 평이한 서두이나 지루하지 않고 주제는 드러남 (Lv.1 평균 수준) |
| 2 | 서두가 건조하여 독자의 관심을 잡지 못함 |
| 1 | 주제조차 드러나지 않는 무의미한 서두, 독자 고려 전혀 없음 |

---

## R11 — Diction (언어 경제성)

> **Lv.1 적용 범위**: 기본 어휘 정확성 + 장황한 풀어쓰기 회피만 평가  
> **평가 질문**: 같은 뜻을 더 짧고 정확한 단어로 쓸 수 있는가?

| 점수 | 기준 |
|------|------|
| 5 | 중학생 수준에서 단어 선택이 정확하고 잉여 수식이 없음 (예: "정말 너무 아주 많이" 같은 반복 없음) |
| 4 | 대부분 적절하나 한두 문장에 장황한 풀이가 남아 있음 |
| 3 | 일상 어휘 중심, 압축할 수 있는 표현이 곳곳에 보이나 의미 전달에 지장 없음 (Lv.1 평균 수준) |
| 2 | 잉여 수식·장황한 풀이가 반복되어 핵심 의미가 흐려짐 |
| 1 | 단어 선택이 모호해서 무엇을 말하는지 파악 어려움 |
