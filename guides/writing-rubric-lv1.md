# Lv.1 채점 루브릭 — 중학생

> **핵심 목표**: 주장과 근거를 구분할 수 있는가  
> **합격선**: R1 Gate Pass + 16/40  
> **기준**: 3점 = 평균적인 중학생 글 수준

---

## R1 — Stance [Gate]

| 판정 | 기준 |
|------|------|
| **Pass** | 한쪽 입장이 글에서 보임 (찬성 또는 반대) |
| **Soft Fail** | 입장이 있으나 모호하거나 글 중간에 바뀜 |
| **Hard Fail** | 주장 자체 없음, 또는 주제와 무관한 내용 |

---

## R2 — Focus (주제 일관성)

| 점수 | 기준 |
|------|------|
| 5 | 모든 문장이 주제와 직접 연결됨. 불필요한 문장 없음 |
| 4 | 1~2문장이 주제와 느슨하게 연결되지만 전체 흐름 유지 |
| 3 | 주제와 관련 없는 문장이 1~2개 포함되나 전체 의도는 파악됨 |
| 2 | 주제에서 벗어난 내용이 전체의 1/3 이상 |
| 1 | 글의 절반 이상이 주제와 무관 |

---

## R3 — Syntax (문법/언어 정확성)

> Lv.1 평가 범위: 맞춤법·띄어쓰기·기본 문장 구조

| 점수 | 기준 |
|------|------|
| 5 | 기초 문법 오류 없음, 문장이 자연스러움 |
| 4 | 오류 1~2개, 의미 전달에 지장 없음 |
| 3 | 오류가 산발적으로 있으나 내용 파악 가능 |
| 2 | 오류 빈번, 독자가 의미를 재해석해야 함 |
| 1 | 기초 문법 오류로 의미 파악 자체가 어려움 |

---

## R4 — Cohesion (유기성/연결성)

> Lv.1 평가 기준: 문장이 순서대로 자연스럽게 이어지는가

| 점수 | 기준 |
|------|------|
| 5 | 문장 간 연결이 자연스럽고 순서가 논리적 |
| 4 | 대체로 자연스러우나 한 곳에서 약간의 비약 |
| 3 | 연결어를 사용하나 실제 논리 연결이 피상적 |
| 2 | 문장 간 연결이 끊기거나 순서가 어색 |
| 1 | 문장이 나열 수준, 흐름 없음 |

---

## R5 — Structure (논리 전개 구조)

> Lv.1 요구 구조: **Claim + Example** (2단계만 요구)

| 점수 | 기준 |
|------|------|
| 5 | 주장과 근거(예시)가 명확히 구분되고 연결됨 |
| 4 | 주장·근거 구조 있으나 한쪽이 약하거나 간략 |
| 3 | 주장 또는 근거 중 하나만 명확히 드러남 |
| 2 | 구조의 흔적은 있으나 주장과 근거의 연결 불분명 |
| 1 | 구조 없음, 생각의 흐름대로 나열 |

---

## R6 — Depth (논거 독립성/심화)

> Lv.1 요구 수준: 논거 1개라도 주장과 연결되면 3점(합격 수준)

| 점수 | 기준 |
|------|------|
| 5 | 논거 2개 이상, 각각 주장과 독립적으로 연결 |
| 4 | 논거 2개이나 하나가 다른 하나의 반복 수준 |
| 3 | 논거 1개, 주장과 명확히 연결됨 |
| 2 | 논거가 있으나 주장과의 연결이 약함 |
| 1 | 논거 없음, 주장만 반복 |

---

## R7 — Flow (가독성/독자 경험)

> Lv.1 평가 기준: 읽다가 막히지 않는 수준

| 점수 | 기준 |
|------|------|
| 5 | 처음부터 끝까지 자연스럽게 읽힘 |
| 4 | 대체로 잘 읽히나 한 곳에서 속도가 느려짐 |
| 3 | 2~3곳에서 멈칫하거나 재독 필요 |
| 2 | 의구심이 자주 발생, 독자가 능동적으로 해석해야 함 |
| 1 | 읽는 중 길을 잃는 느낌 |

---

## R8 — Paraphrasing (표현 다양성)

> Lv.1 적용 범위: **B만** — 동일 표현 반복 없이 다양하게 전달하는가

| 점수 | 기준 |
|------|------|
| 5 | 같은 개념·주장을 반복 없이 다양한 표현으로 전달 |
| 4 | 표현 반복 1~2회, 의미 전달에 지장 없음 |
| 3 | 같은 표현이 반복되어 어휘 범위가 좁게 느껴짐 |
| 2 | 핵심 표현·문장이 문단 전체에 걸쳐 반복됨 |
| 1 | 주제 문장이나 주장을 그대로 복사·반복 |

---

## R9 — Universality (논거·근거의 보편성)

> Lv.1 적용 범위: 개인 사례 허용. **보편성 인식 여부**만 확인

| 점수 | 기준 |
|------|------|
| 5 | 개인 경험을 넘어 보편적 적용 가능성을 명시적으로 언급 |
| 4 | 개인 사례이지만 "많은 사람도..." 식의 보편화 시도가 보임 |
| 3 | 개인 사례만 제시 (Lv.1 기준 합격 수준) |
| 2 | 개인 감정·주관적 의견만, 구체적 사례 없음 |
| 1 | 근거 자체 없음 |

---
