# 논리적 글쓰기 채점 루브릭

---

## 채점 체계 개요

| 단계 | 기준 | 방식 |
|------|------|------|
| **Gate** | R1 Stance | Pass / Soft Fail / Hard Fail — Hard Fail 시 채점 중단 |
| **Score** | R2~R11 | 각 5점 만점 × 10개 = **50점 만점** |

**평가 축 개요**

| 축 | 기준 | 성격 |
|----|------|------|
| 논리·구조 | R2 Focus, R5 Structure, R6 Depth, R9 Universality | 글의 뼈대 |
| 언어 정확성 | R3 Syntax, R4 Cohesion, R8 Paraphrasing | 문장 단위 정오 |
| 독자 경험 | R7 Flow | 읽힘 장애물 부재 |
| 표현력 | **R10 Rhetoric, R11 Diction** | 효과와 효율 — 문장 단위의 설득력·경제성 |

---

## R1 — Stance (입장 명확성) [Gate]

주장이 글 전체를 지탱하는 기둥이다. 이게 흔들리면 나머지 채점이 무의미하다.

| 판정 | 기준 | 처리 |
|------|------|------|
| **Pass** | 주장이 명확하고 한 문장으로 정리됨 | R2~R11 채점 진행 |
| **Soft Fail** | 주장은 있으나 모호하거나 양면적 | R1 피드백만 제공 → 재제출 → R2~R11 채점 진행 |
| **Hard Fail** | 주장 자체가 없거나 주제와 무관 | 채점 중단 → 전면 재작성 요구 |

**레벨별 Pass 기준 (요약):**

| 레벨 | Pass 기준 |
|------|----------|
| Lv.1 | 찬성/반대 중 한쪽 입장이 글 어딘가에 보임 |
| Lv.2 | 주장이 한 문장으로 정리되고 입장이 명확함 |
| Lv.3 | Thesis가 서론(첫 문단)에 명시되고 한 문장으로 정리됨 |
| Lv.4 | Thesis + 앞으로 전개할 논거의 방향이 서론에 함께 제시됨 |
| Lv.5 | Thesis + 논거 방향 + 적용 조건·범위 한정(Qualifier) |
| Lv.6 | 핵심 결론 + Recommendation이 서두(Executive summary)에 제시 (BLUF 축) |
| Lv.7 | Thesis + 학술적 기여(contribution) + 선행 연구와의 차별점 |

**Lv.6 주의**: Lv.6은 학술 논증이 아니라 의사결정 지원 문서이므로 "Thesis"가 아닌 "결론+Recommendation" 축으로 평가한다. 다른 레벨의 Thesis 중심 기준을 기계적으로 적용하지 않는다.

**상세 anchor**는 각 `writing-rubric-lv{N}.md`에서 정의되며, 본 요약과 불일치 시 레벨별 파일이 우선한다.

## R2 — Focus (주제 일관성)

| 점수 | 기준 |
|------|------|
| 5 | 모든 문장이 thesis에 직접 기여, 불필요한 문장 없음 |
| 4 | 1~2문장이 주변부적이나 전체 흐름에 영향 없음 |
| 3 | 한 문단이 thesis와 느슨하게 연결됨 |
| 2 | 주제에서 벗어난 문단이 존재, 독자가 혼란 |
| 1 | 글의 절반 이상이 thesis와 무관 |

---

## R3 — Syntax (문법/언어 정확성)

**모드에 따라 평가 기준이 달라진다.**

| 모드 | 적용 조건 | R3 평가 대상 |
|------|----------|------------|
| **일반 모드** | 퇴고 가능한 글 (리포트, 논문, 보고서) | 오타·띄어쓰기·문법·어휘 모두 |
| **Timed 모드** | 시간 제한 조건 (TOEFL, GRE, 논술 시험, 세션 내 즉석 작성) | 문법 구조 다양성 + 어휘 **범위(Range)**만. 오타·띄어쓰기 감점 없음 |

**일반 모드 기준:**

| 점수 | 기준 |
|------|------|
| 5 | 오류 없음, 어휘 선택이 정확하고 자연스러움 |
| 4 | 사소한 오류 1~2개, 의미 전달에 영향 없음 |
| 3 | 오류가 산발적으로 있으나 의미는 파악 가능 |
| 2 | 오류가 빈번하여 독자가 의미를 재해석해야 함 |
| 1 | 오류가 글 전반에 퍼져 의미 전달 자체가 어려움 |

**Timed 모드 기준:**

| 점수 | 기준 |
|------|------|
| 5 | 다양한 문법 구조 구사, 고급 어휘 정확하게 사용 |
| 4 | 문법 구조 다양, 어휘 선택 대체로 적절 |
| 3 | 단순 문장 구조 반복, 어휘 범위가 좁음 |
| 2 | 문법 오류가 의미 전달을 방해 |
| 1 | 기초 문법 오류로 의미 파악 어려움 |

*레벨별 적용: Lv.1~2는 기초 문법, Lv.5~7은 어휘 정확성·어조(tone) 포함*

**R11과의 경계선**: R3는 어휘의 **범위(Range)**, R11은 어휘의 **정밀도·경제성(Precision)**을 본다. 다양한 고급 단어를 쓰지만 모두 장황하면 R3는 고득점, R11은 저득점이다. 반대로 어휘 범위는 좁아도 각 단어가 정확하게 압축적이면 R3는 낮고 R11은 높다. Timed 모드에서는 R3가 범위를, R11이 정밀도를 각각 담당한다.

## R4 — Cohesion (유기성/연결성)

| 점수 | 기준 |
|------|------|
| 5 | 문장↔문장, 문단↔문단이 자연스럽게 이어짐. 논리 자체로 흐름 형성 |
| 4 | 연결이 대체로 자연스러우나 한 곳에서 약간의 비약 |
| 3 | 연결어는 있으나 논리적 연결이 피상적, 문단 간 점프가 느껴짐 |
| 2 | 문단 단위 연결이 끊기거나 순서가 부자연스러움. 문장 단위는 유지 |
| 1 | 문장 단위부터 연결이 부재하여 독립된 문장·구절의 나열처럼 읽힘 |

**평가 주의사항**: 숙련된 글쓰기에서 단락 전환은 명시적 연결어("다음으로", "또한") 없이 **전 단락 마지막 문장이 다음 단락을 자연스럽게 예고**하는 방식으로 이루어진다. 연결어 부재를 곧 R4 감점 사유로 처리하지 않는다. 논리의 흐름이 끊기는지 여부로 판단한다.

---

## R5 — Structure (논리 전개 구조)

> **평가 원칙**: 해당 레벨이 요구하는 논증 구조가 완성되었는지, 각 단계가 명확히 구분되고 기여하는지를 본다. 구체적 구조 정의는 레벨마다 다르므로 **레벨별 anchor가 우선**이다.

| 점수 | 기준 |
|------|------|
| 5 | 레벨이 요구하는 모든 구조 단계가 완성되고, 각 단계가 명확히 구분되며 자기 역할을 수행한다 |
| 4 | 요구 구조가 있으나 한 단계가 약하거나 간략하게 처리됨 |
| 3 | 일부 단계가 누락되거나 전개가 불균형하여 한 레벨 아래 수준으로 읽힘 |
| 2 | 구조의 흔적은 있으나 전개 방식이 불분명하고 단계 구분이 흐림 |
| 1 | 구조 없음, 생각의 흐름대로 나열 |

**레벨별 요구 구조 (레벨 파일 참조)**:

| 레벨 | 요구 구조 |
|------|----------|
| Lv.1 | Claim + Example (2단계) |
| Lv.2 | Claim → Reasoning → Example (3단계) |
| Lv.3 | 서론(Thesis) → 본론1 → 본론2 → 반론 인식 → 결론 (5단락) |
| Lv.4 | 서론(Thesis+논거방향) → 본론1+evidence → 본론2+evidence → 반론+반박 → 결론 |
| Lv.5 | Toulmin — Claim → Grounds → Warrant → Qualifier → Rebuttal |
| Lv.6 | BLUF — Executive summary → 근거·분석 → 리스크·제약 → Action items |
| Lv.7 | IMRaD — Introduction(gap+contribution) → Methodology → Results → Discussion → Conclusion |

## R6 — Depth (논거 독립성/심화)

| 점수 | 기준 |
|------|------|
| 5 | Supporting idea 2개 이상, 독립적이거나 심화 종속적. 각 논거가 thesis를 다른 각도에서 강화 |
| 4 | 논거 2개 이상이나 한 논거가 다른 논거의 반복 수준 |
| 3 | 논거가 1개이거나 2개지만 사실상 동일한 주장 |
| 2 | 논거가 있으나 주장을 뒷받침하지 못하고 표면적 |
| 1 | 논거 없음, 주장만 반복 |

## R8 — Paraphrasing (표현 다양성 / 재진술 능력)

| 점수 | 기준 |
|------|------|
| 5 | 같은 주장·개념을 반복 없이 다양한 표현으로 전달. Lv.4 이상: 외부 주장·근거를 자신의 언어로 정확히 재진술 |
| 4 | 표현 반복이 1~2회 있으나 의미 전달에 영향 없음 |
| 3 | 같은 표현이 반복되어 어휘 범위가 좁게 느껴짐. Lv.4 이상: 외부 내용을 거의 그대로 옮김 |
| 2 | 핵심 표현이 문단 전체에 걸쳐 그대로 반복됨 |
| 1 | 주제 문장이나 주장을 그대로 복사·반복 |

*레벨별 적용: Lv.1~2는 B(다양한 표현·반복 회피)만 적용 / Lv.3~4는 B+A(자신의 언어로 재진술) / Lv.5~7은 B+A+C(외부 주장·근거 요약·변환 포함)*

---

## R9 — Universality (논거·근거의 보편성)

| 점수 | 기준 |
|------|------|
| 5 | 논거가 보편적 원리·데이터·사회적 사실에 기반, 폭넓은 설득력 |
| 4 | 보편적 근거 주도, 개인 사례는 보조로만 사용 |
| 3 | 개인 사례와 보편 근거가 혼재, 설득력이 들쭉날쭉 |
| 2 | 대부분 개인 경험·주관적 판단에 의존 |
| 1 | 논거 전체가 개인 경험 수준, 일반화 불가 |

*레벨별 적용: Lv.1~2는 개인 사례 허용, 보편성 인식 여부만 확인 / Lv.3~4는 보편 근거 1개 이상 요구 / Lv.5~7은 논거 대부분이 보편 근거여야 Pass 수준*

## R7 — Flow (가독성/독자 경험)

> **평가 축**: 독자가 **멈추거나 재독할 필요 없이** 매끄럽게 읽을 수 있는가. 읽기의 **장애물 부재**를 본다.
> **R10과의 경계선**: 독자가 능동적으로 끌려가는지(설득력)는 R10 Rhetoric의 영역이다. R7은 단지 "막히지 않는가"만 평가한다. 두 기준을 혼동해 중복 감점하지 않는다.

| 점수 | 기준 |
|------|------|
| 5 | 처음부터 끝까지 멈춤 없이 읽히고 재독할 필요가 없음 |
| 4 | 대체로 매끄럽게 읽히나 한 곳에서 속도가 느려짐 |
| 3 | 2~3곳에서 멈칫하거나 재독이 필요 |
| 2 | 의구심이 자주 발생하여 독자가 능동적으로 해석해야 함 |
| 1 | 읽는 중 길을 잃어 전체 재독이 필요 |

**평가 주의사항 — Hooking Sentence**: 서론 첫 문장을 통계·과학적 사실·항목 열거 등으로 시작해 독자의 이목을 잡는 테크닉은 TOEFL·GRE·학술 글쓰기의 표준 기법이다(Lv.4 이상). 배경 설명 없이 바로 시작하는 것처럼 보여도 hook → thesis 연결이 의도적 구조임을 인식하고 R7 감점 사유로 처리하지 않는다.

## R10 — Rhetoric (수사 효과)

> **평가 축**: 정보 전달의 **효과성**. 독자를 능동적으로 끌어당기고 움직이는 설득력.
> **평가 대상**: Hook 구성, 대조·병렬·강조 등 수사 장치, tone modulation, 독자 인식(audience awareness), pathos/logos/ethos 배합.

| 점수 | 기준 |
|------|------|
| 5 | Hook + 대조·병렬·강조 등 수사 장치가 thesis 강화를 위해 의도적으로 배치되어 독자를 능동적으로 끌어당긴다 |
| 4 | 주요 수사 장치 한두 개가 효과적으로 작동하나 thesis와의 연결이 한 곳에서 부분적 |
| 3 | 수사 장치 사용이 간헐적이거나 표면적이어서 임팩트가 약하다 |
| 2 | 수사 장치가 거의 없고 단조로운 진술이 이어져 독자의 관심을 잡지 못한다 |
| 1 | 평이한 나열에 그쳐 독자의 반응을 유도할 장치가 전혀 없다 |

**R7과의 경계선**: R7 Flow는 "읽다가 막히지 않는가"라는 **수동적** 기준(장애물 부재)이고, R10 Rhetoric은 "읽고 끌려가는가"라는 **능동적** 기준(설득력)이다. 읽기 장애물이 없다고 해서 설득력이 있는 것은 아니다. 두 기준을 혼동해 중복 감점하지 않는다.

**레벨별 적용**: Lv.1~2는 서두 흥미 유발 여부 정도만 평가, Lv.3~4는 hook + 기초 수사 장치, Lv.5~7은 pathos/logos/ethos 배합과 tone modulation까지 평가.

---

## R11 — Diction (언어 경제성)

> **평가 축**: 정보 전달의 **효율성**. 같은 의미를 더 짧고 정확한 단어로 전달하는 능력.
> **평가 대상**: 단어 선택의 정밀도, 긴 풀이를 한 단어로 압축하는 능력, 잉여 수식어 제거, 레지스터 일관성.
> **앵커 예시**: "내가 가진 모든 자원과 에너지를 활용하여 목표 성취를 위해 최선을 다한다" → "**총력**으로 임한다". 이런 압축 사례가 반복되면 5점.

| 점수 | 기준 |
|------|------|
| 5 | 긴 설명을 함축하는 정확한 단어를 반복적으로 구사하고 잉여 수식이 없다. 레지스터가 일관된다 |
| 4 | 대체로 정밀한 어휘를 선택하나 한두 곳에서 장황한 풀이가 남아 있다 |
| 3 | 일상적 어휘가 지배적이며 압축할 수 있는 표현이 곳곳에 보인다 |
| 2 | 잉여 수식·장황한 풀이가 반복되어 핵심 의미가 흐려진다 |
| 1 | 단어 선택이 모호하거나 부정확해서 무엇을 말하는지 파악이 어렵다 |

**R3·R8과의 경계선**:
- R3 Syntax는 "어휘가 **틀렸는가**"라는 정오(正誤) 기준이고, R11 Diction은 "더 **좋은** 단어가 있는가"라는 최적성 기준이다. 오류 없는 일상 어휘는 R3에서 감점 사유가 아니지만 R11에서는 3점 이하로 처리한다.
- R8 Paraphrasing은 "같은 표현을 **반복**하지 않는가"라는 다양성 기준이고, R11 Diction은 "긴 표현을 한 단어로 **압축**하는가"라는 경제성 기준이다. 서로 다른 표현을 여러 번 써도 모두 장황하면 R8은 고득점, R11은 저득점일 수 있다.

**Timed 모드 주의사항**: 시간 제약에서는 최적 어휘 탐색이 어려우므로 R11 감점을 1점 범위 내로 완화한다. 반면 R10은 TOEFL·GRE가 적극적으로 평가하는 항목이므로 Timed 모드에서도 그대로 적용한다.

**레벨별 적용**: Lv.1~2는 기본 어휘 정확성과 장황함 회피만 평가, Lv.3~4는 압축 시도 여부 + 잉여 수식 제거, Lv.5~7은 고급 어휘의 경제적 구사 + 레지스터 일관성까지 평가.

## 피드백 우선순위

| 티어 | 기준 | 피드백 활성화 조건 |
|------|------|-----------------|
| 1순위 | R2 Focus, R5 Structure | 항상 활성 |
| 2순위 | R3 Syntax, R4 Cohesion, R6 Depth, R8 Paraphrasing, R9 Universality | R2·R5 Resolved 후 활성 |
| 3순위 | R7 Flow, R10 Rhetoric, R11 Diction | R2~R6, R8~R9 전체 Resolved 후 활성 |

**피드백 상한**: 세션당 최대 3개. 가장 낮은 점수 기준 우선 티어 순으로 선정.

**R10·R11 우선순위 주의**: 논리·구조가 먼저다. R2 Focus나 R5 Structure가 낮은 글에 "더 좋은 단어 없을까?"를 먼저 지적하면 학습자의 우선순위가 왜곡된다. R10/R11은 반드시 3순위로 유지한다.

## 레벨별 합격선

| 레벨 | 대상 | 합격선 |
|------|------|--------|
| Lv.1 | 중학생 | R1 Gate + 38/50 |
| Lv.2 | 고등학생 | R1 Gate + 38/50 |
| Lv.3 | 수능 논술 | R1 Gate + 38/50 |
| Lv.4 | 대학 리포트 / TOEFL | R1 Gate + 38/50 |
| Lv.5 | GRE AW | R1 Gate + 38/50 |
| Lv.6 | 실무 보고서 | R1 Gate + 38/50 |
| Lv.7 | 학술 논문 | R1 Gate + 38/50 |

**합격선 고정 원리**: 모든 레벨에서 38/50 = **75%**로 통일한다. 이유: R2~R11의 1~5점 anchor가 이미 각 레벨 스펙트럼 내에서 정의되므로(예: Lv.1의 5점 = "중학생 최고 수준", Lv.7의 5점 = "peer-review 통과 수준"), 레벨 간 상대 난이도는 anchor 자체에 내장되어 있다. 여기에 합격 %까지 차등화하면 이중 가중이 되어 "Lv.7은 거의 만점을 받아야 통과"라는 왜곡이 발생한다.

75%의 의미: 10개 기준에서 평균 3.8점 — "모든 축에서 해당 레벨의 평균을 안정적으로 상회"라는 기준.

## 레벨 자동 감지

글 제출 시 선언 레벨과 실제 수준이 다르면 재채점을 제안한다.

| 상황 | 처리 |
|------|------|
| 실제 수준 > 선언 레벨 | "실제 수준 Lv.N으로 재채점을 제안한다" + 재채점 결과 병기 |
| 실제 수준 < 선언 레벨 | 선언 레벨 기준 채점 유지, 부족한 부분 명시 |
| 일치 | 그대로 진행 |

**판단 기준**:
- **Primary**: R6 Depth 5점 + R5 Structure 4점 이상이면 선언 레벨보다 1~2단계 높은 수준으로 판단한다.
- **Secondary (참고)**: R10 Rhetoric 또는 R11 Diction이 5점이면 표현력 축에서 상위 레벨 특성이 감지되므로 Primary 판단의 확신도가 상승한다. 단 표현력만 높고 논리·구조가 낮으면 "문장력은 높으나 논증이 부족한 글"로 기록하고 레벨 조정을 보류한다.
