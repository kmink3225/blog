## Chapter Summaries

### Ch 1: Introduction (pp. 1-30)

**핵심**: 머신러닝이 무엇인지 정의하고 주요 유형을 소개한다. 지도학습(분류, 회귀, 과적합과 일반화, 무공짜 점심 정리), 비지도학습(군집화, 잠재 인자 발견, 자기지도학습), 강화학습을 설명한다. 일반적인 이미지/텍스트 데이터셋과 데이터 전처리 방법도 다룬다.

**키워드**: `classification`, `regression`, `overfitting`, `no free lunch`, `self-supervised learning`

**상세**: → `Probabilistic Machine Learning An Introduction_azure_full.md` Ch 1 (line 115)

머신러닝을 데이터로부터 패턴을 학습하여 예측하는 방법으로 정의하고, 지도학습, 비지도학습, 강화학습의 세 가지 주요 유형을 소개한다. 지도학습을 분류(이산 출력)와 회귀(연속 출력)로 구분하며, 이미지 분류(MNIST, CIFAR, ImageNet)와 텍스트 분류(감성 분석) 등의 실제 예시를 제시한다. 과적합과 일반화 개념을 설명하며, 훈련 오차와 테스트 오차의 차이를 통해 모델 복잡도가 일반화에 미치는 영향을 논한다. 무공짜 점심 정리를 소개하여 모든 문제에 대해 최적인 단일 알고리즘은 존재하지 않으며, 데이터에 적합한 귀납적 편향 선택이 핵심임을 강조한다. 비지도학습에서 군집화, 잠재 인자 발견, 자기지도학습을 설명하며, 생성 모형과 판별 모형의 구분을 소개한다. 자기지도학습이 레이블 없는 데이터로부터 유용한 표현을 학습하는 현대적 패러다임임을 설명한다. 강화학습에서 에이전트-환경 상호작용, 보상 신호, 탐색-활용 트레이드오프의 기본 개념을 다룬다. 일반적인 이미지 데이터셋(MNIST, Fashion-MNIST, CIFAR, ImageNet)과 텍스트 데이터셋을 소개한다. 이산 입력의 전처리(원-핫 인코딩, 특징 교차)와 텍스트 데이터 전처리(토큰화, 단어 주머니, TF-IDF, 단어 임베딩)를 다룬다. 결측 데이터 처리 방법을 소개하며 머신러닝과 통계학, 데이터 마이닝, 인공지능 등 관련 분야와의 관계를 논한다.

### Ch 2: Probability: Univariate Models (pp. 33-76)

**핵심**: 일변량 확률 모형을 다룬다. 확률의 정의와 불확실성 유형, 확률변수(이산/연속, 적률), 베이즈 정리(COVID-19 검사 예제, 몬티 홀 문제), 베르누이/이항 분포, 범주형/다항 분포(소프트맥스 함수), 가우시안 분포, 기타 분포(t-분포, 라플라스, 베타, 감마)를 설명한다.

**키워드**: `Bayes rule`, `Bernoulli`, `categorical`, `Gaussian`, `softmax`, `Student-t`

**상세**: → `Probabilistic Machine Learning An Introduction_azure_full.md` Ch 2 (line 139)

확률의 정의를 빈도론적 해석과 베이지안(주관적) 해석 두 관점에서 소개하고, 인식론적 불확실성과 우연적 불확실성을 구분한다. 이산 확률변수의 확률질량함수와 연속 확률변수의 확률밀도함수, 누적분포함수를 정의하며, 분위수와 적률(평균, 분산, 최빈값, 조건부 적률)을 설명한다. 베이즈 정리를 사전확률에서 사후확률로의 갱신 규칙으로 해석하며, COVID-19 검사의 민감도·특이도 예제를 통해 위양성 역설을 보인다. 몬티 홀 문제를 베이즈 정리로 풀어 조건부 확률의 반직관적 결과를 시연한다. 베르누이 분포와 이항 분포를 정의하고, 시그모이드(로지스틱) 함수가 로그오즈를 확률로 변환하는 역할을 설명한다. 범주형 분포와 다항 분포를 소개하며, 소프트맥스 함수가 실수 벡터를 확률 심플렉스로 매핑하는 과정과 로그-합-지수 트릭의 수치적 안정성을 다룬다. 가우시안 분포의 PDF, CDF, 중심극한정리에 의한 정당화를 설명하고, 디랙 델타 함수와 절단 가우시안을 극한 사례로 다룬다. 스튜던트 t-분포의 두꺼운 꼬리에 의한 이상치 강건성, 라플라스 분포, 베타 분포, 감마 분포 등의 성질을 소개한다. 확률변수의 변환에서 야코비안을 이용한 밀도 변환 규칙, 선형 변환의 적률 변환, 합성곱 정리를 유도한다. 중심극한정리를 형식적으로 기술하고 몬테카를로 근사의 기본 원리를 제시한다.

### Ch 3: Probability: Multivariate Models (pp. 77-106)

**핵심**: 다변량 확률 모형을 다룬다. 결합 분포, 주변 분포, 조건부 분포, 다변량 가우시안, 혼합 모형, 확률적 그래프 모형(DAG, 조건부 독립)을 설명한다.

**키워드**: `joint distribution`, `multivariate Gaussian`, `mixture models`, `graphical models`

**상세**: → `Probabilistic Machine Learning An Introduction_azure_full.md` Ch 3 (line 192)

결합 분포, 주변 분포, 조건부 분포의 관계를 다루며 공분산과 상관계수의 정의와 한계를 설명한다. 비상관이 독립을 함의하지 않는 예를 제시하고, 상관이 인과를 함의하지 않음을 심슨의 역설을 통해 경고한다. 다변량 가우시안(MVN) 분포를 정의하며, 마할라노비스 거리가 타원체 등고면을 결정하고 공분산 행렬의 고유값 분해가 타원의 축과 크기를 결정함을 시각화한다. MVN의 주변 분포와 조건부 분포가 다시 가우시안이 되는 성질을 유도하며, 2차원 예제를 통해 조건부 평균이 선형 함수, 조건부 분산이 조건 변수에 독립임을 확인한다. 결측 데이터 대체(imputation)에의 응용을 조건부 가우시안을 이용하여 설명한다. 선형 가우시안 시스템에서 가우시안에 대한 베이즈 규칙을 유도하여 사전 가우시안과 선형 가우시안 우도의 결합이 가우시안 사후를 산출함을 보인다. 미지 스칼라 및 벡터의 추론, 센서 융합 예제를 통해 정밀도의 가법성을 보인다. 지수족 분포의 일반적 정의를 제시하고, 로그 분할 함수가 큐뮬런트 생성 함수 역할을 하며 최대 엔트로피 원리로부터 지수족이 유도됨을 설명한다. 혼합 모형을 도입하여 가우시안 혼합과 베르누이 혼합으로 다봉 분포를 표현하는 방법을 다룬다. 확률적 그래프 모형의 기초를 소개하며 DAG 표현, 조건부 독립, 플레이트 표기법 등을 설명한다.

### Ch 4: Statistics (pp. 107-166)

**핵심**: 통계적 추론의 기초를 다룬다. 최대우도추정(MLE), 경험적 위험 최소화(ERM), 모형의 과적합/부적합, 정규화, 베이지안 통계(사전분포, 사후분포, 사후예측, 신뢰구간), 빈도론적 통계(편향-분산 트레이드오프, 신뢰구간, 가설검정)를 설명한다.

**키워드**: `MLE`, `ERM`, `Bayesian statistics`, `frequentist statistics`, `regularization`

**상세**: → `Probabilistic Machine Learning An Introduction_azure_full.md` Ch 4 (line 224)

최대우도추정(MLE)을 정의하고, KL 발산 최소화 및 경험적 위험 최소화와의 동치성을 통해 정당화한다. 베르누이, 범주형, 일변량·다변량 가우시안, 선형 회귀에 대한 MLE를 구체적으로 유도한다. 경험적 위험 최소화(ERM)에서 대리 손실 함수의 필요성을 오분류율의 미분 불가능성으로 설명하고, 교차 엔트로피·힌지 손실 등을 비교한다. 모멘트 방법과 온라인(순차적) 추정을 대안적 추정 방법으로 소개한다. 정규화를 과적합 방지의 핵심 도구로 소개하며, MAP 추정이 사전분포에 대응하는 벌칙 항을 추가한 최대우도와 동치임을 보인다. 검증 집합과 교차검증을 이용한 정규화 강도 선택, 조기 종료, 더 많은 데이터 확보의 효과를 논한다. 베이지안 통계에서 켤레 사전분포를 중심으로 베타-이항, 디리클레-다항, 가우시안-가우시안 모형의 사후분포 도출을 상세히 전개한다. 사후예측분포가 매개변수 불확실성을 통합하여 MLE보다 과적합에 강건함을 보이며, 신뢰구간(크레디블 구간)의 의미를 설명한다. 빈도론적 통계에서 표집 분포, MLE의 가우시안 근사, 부트스트랩, 신뢰구간을 다루며, 신뢰구간과 크레디블 구간의 해석적 차이를 강조한다. 편향-분산 트레이드오프를 기대 손실의 분해로 형식화하고, 모델 복잡도에 따른 편향과 분산의 상반관계를 시각적으로 보인다.

### Ch 5: Decision Theory (pp. 167-206)

**핵심**: 결정이론을 다룬다. 베이지안 결정이론, 분류기 평가(ROC, PR 곡선), 회귀 손실, 확률적 예측(적절 채점 규칙), 불확실성 하의 결정(효용이론, 다기준 결정 분석)을 설명한다.

**키워드**: `Bayesian decision theory`, `ROC`, `loss functions`, `proper scoring rules`

**상세**: → `Probabilistic Machine Learning An Introduction_azure_full.md` Ch 5 (line 83)

베이지안 결정이론의 기본 프레임워크를 도입하여 손실 함수와 사후분포를 결합하여 최적 행동을 선택하는 원리를 설명한다. 분류 문제에서 0-1 손실 하의 최적 결정이 사후확률 최대 클래스 선택과 동치임을 보이고, 비대칭 손실(의료 진단 등)의 처리를 다룬다. ROC 곡선과 AUC를 이용한 이진 분류기 평가를 설명하며, 클래스 불균형 시 정밀도-재현율(PR) 곡선과 F1 점수가 더 적합함을 논한다. 회귀 문제에서 L2 손실의 최적해가 조건부 평균, L1 손실의 최적해가 조건부 중앙값임을 유도한다. 확률적 예측의 평가를 위한 적절 채점 규칙(로그 손실, 브라이어 점수)과 보정(calibration)의 중요성을 설명한다. 베이지안 가설 검정에서 사후확률을 이용한 모형 비교를 설명하고, 베이즈 인수와 한계우도의 역할을 다룬다. 오컴의 면도날이 한계우도에 자연스럽게 내장되어 더 단순한 모형을 선호하는 기제를 보인다. 교차검증과 한계우도의 연결, AIC와 BIC 등 정보 기준을 비교하며 모형 선택의 실용적 방법을 제시한다. 빈도론적 결정이론에서 위험 함수, 일치 추정량, 허용 추정량의 개념을 소개하고, 구조적 위험 최소화와 VC 차원을 통한 일반화 한계를 다룬다. 빈도론적 가설 검정(NHST)의 p-값 해석과 그 한계를 비판적으로 논하며 베이지안 대안과 비교한다.

### Ch 6: Information Theory (pp. 207-228)

**핵심**: 정보이론을 다룬다. 엔트로피, 교차 엔트로피, KL 발산, 상호정보량, 데이터 처리 부등식, 충분통계량을 설명한다.

**키워드**: `entropy`, `cross-entropy`, `KL divergence`, `mutual information`

**상세**: → `Probabilistic Machine Learning An Introduction_azure_full.md` Ch 6 (line 84)

이산 확률변수의 엔트로피 H(X) = −Σ p(x)log p(x)를 불확실성의 측도로 정의하고, 균일 분포에서 최대가 됨을 보인다. 교차 엔트로피 H(p,q) = −Σ p(x)log q(x)를 분포 p에 대해 q를 코딩에 사용할 때의 평균 비트 수로 해석하며, 머신러닝에서 손실 함수로 널리 사용됨을 설명한다. 결합 엔트로피와 조건부 엔트로피의 정의 및 체인 규칙 H(X,Y) = H(X) + H(Y|X)를 유도한다. 퍼플렉서티를 언어 모형의 평가 지표로 소개하며 엔트로피와의 관계를 설명한다. 연속 확률변수에 대한 미분 엔트로피를 정의하고 가우시안이 주어진 분산에서 최대 엔트로피 분포임을 보인다. KL 발산의 정의, 비음수성(깁스 부등식), 비대칭성을 다루며, 전방 KL과 역방 KL의 차이를 모드 탐색 대 모드 포착 행동으로 설명한다. KL 발산의 최소화가 MLE와 동치임을 유도하여 정보이론과 통계적 추정의 연결을 보인다. 상호정보량 I(X;Y) = KL(p(x,y)||p(x)p(y))를 두 변수 간의 통계적 의존성 측도로 정의하고, H(X)−H(X|Y)와의 동치성을 보인다. 조건부 상호정보량, 정규화 상호정보량, 최대 정보 계수(MIC) 등의 확장을 소개하며 비선형 의존성 탐지에의 활용을 논한다. 데이터 처리 부등식과 충분통계량, 파노 부등식을 다루어 정보이론의 기본 한계를 제시한다.

LU, QR, 촐레스키 분해와 선형 방정식 풀이(정방·과소결정·과결정 시스템)를 설명한다. 행렬 미적분에서 스칼라-벡터 미분(그래디언트), 야코비안, 헤시안의 정의와 자주 사용되는 행렬 미분 공식을 정리한다.

### Ch 8: Optimization (pp. 275-320)

**핵심**: 최적화 방법을 다룬다. 1차 방법(경사하강법, 모멘텀, SGD, Adam), 2차 방법(뉴턴법, BFGS), 제약 최적화(라그랑주 승수법, KKT 조건), 볼록 최적화, 확률적 최적화를 설명한다.

**키워드**: `gradient descent`, `SGD`, `Adam`, `Newton method`, `convex optimization`

**상세**: → `Probabilistic Machine Learning An Introduction_azure_full.md` Ch 8 (line 395)

최적화의 기본 개념으로 국소 최적 대 전역 최적, 제약 대 비제약, 볼록 대 비볼록, 매끄러운 대 비매끄러운 최적화를 구분한다. 1차 방법에서 하강 방향과 스텝 크기(학습률)의 선택을 논하며, 정확한 직선 탐색과 아르미호·울프 조건을 소개한다. 모멘텀 방법(중모멘텀, 네스테로프 가속 경사)이 좁은 골짜기에서의 진동을 완화하는 원리를 설명한다. 2차 방법으로 뉴턴법의 이차 수렴 특성과 BFGS 등 준뉴턴법이 헤시안을 점진적으로 근사하는 방법, 신뢰영역 방법을 다룬다. 확률적 경사하강(SGD)을 유한 합 문제에 적용하며 미니배치의 역할, 학습률 스케줄(웜업, 코사인 감쇠, 사이클링), 반복 평균(폴리악 평균)을 설명한다. 분산 감소 기법(SVRG, SAGA)과 전처리된 SGD(AdaGrad, RMSProp, Adam, AdamW)의 원리를 비교하며, Adam이 실무에서 널리 쓰이는 이유를 논한다. 제약 최적화에서 라그랑주 승수법과 KKT 조건을 유도하고, 선형 프로그래밍과 이차 프로그래밍의 기본을 다룬다. 근접 경사법(proximal gradient)을 L1 정규화의 비매끄러운 최적화에 적용하며 소프트 임계화 연산자를 유도한다. 한계 최적화(bound optimization)의 일반 프레임워크를 소개하고 EM 알고리즘을 그 특수한 경우로 유도하여 GMM 학습 예제를 제시한다. 블랙박스 및 도함수 없는 최적화(그리드 탐색, 랜덤 탐색, 베이지안 최적화)를 초매개변수 튜닝에 활용하는 방법을 논한다.

### Ch 9: Linear Discriminant Analysis (pp. 323-338)

**핵심**: 선형 판별분석을 다룬다. 가우시안 판별분석(GDA), 이차 판별분석(QDA), 정규화 판별분석, 최근접 축소 중심(NSC), 피셔 선형 판별을 설명한다.

**키워드**: `GDA`, `QDA`, `Fisher discriminant`, `nearest shrunken centroids`

**상세**: → `Probabilistic Machine Learning An Introduction_azure_full.md` Ch 9 (line 439)

가우시안 판별분석(GDA)을 도입하여 각 클래스의 조건부 분포가 가우시안일 때 베이즈 정리를 통해 결정경계를 유도한다. 이차 판별분석(QDA)은 클래스별 별도의 공분산 행렬을 사용하여 이차 결정경계를 생성하고, 공분산을 공유하면 선형 판별분석(LDA)으로 단순화되어 선형 결정경계가 된다. LDA와 로지스틱 회귀의 연결을 보이며 공유 가우시안 가정 하에서 사후확률이 로지스틱 함수의 선형 결합으로 표현됨을 유도한다. MLE를 통한 GDA 매개변수 추정—클래스별 평균, 공유 공분산, 사전확률—의 폐쇄형 해를 제시한다. 정규화 판별분석(RDA)이 QDA와 LDA 사이를 보간하여 과적합을 제어하는 방법과 최근접 축소 중심(NSC)이 각 클래스 평균을 전체 평균 쪽으로 축소하여 특징 선택 효과를 달성하는 방법을 설명한다. 피셔 선형 판별을 클래스 간 분산 대 클래스 내 분산 비의 최대화로 유도하며, 차원 축소와 분류를 동시에 수행하는 관점을 제시한다. 나이브 베이즈 분류기를 조건부 독립 가정 하의 생성 모형으로 소개하고, 가우시안·베르누이·범주형 특징에 대한 변형을 다룬다. 나이브 베이즈의 MLE와 베이지안 학습(라플라스 평활)을 설명하고, 나이브 베이즈와 로지스틱 회귀의 생성-판별 관계를 논한다. 생성 분류기와 판별 분류기의 장단점을 비교하며, 생성 모형이 결측 특징 처리와 적은 데이터에 유리하고 판별 모형이 더 유연한 결정경계를 학습함을 설명한다. 결측 특징 처리 시 생성 모형이 주변화를 통해 자연스럽게 대처하는 방법을 다룬다.

**핵심**: 로지스틱 회귀를 다룬다. 이진/다항 로지스틱 회귀, MLE 추정(SGD, 뉴턴법), 정규화(L2, L1), 베이지안 로지스틱 회귀(라플라스 근사, 변분추론)를 설명한다.

이진 로지스틱 회귀를 시그모이드 함수를 통한 선형 분류 모형으로 정의하고, 비선형 기저함수(다항식, RBF)를 통해 비선형 결정경계로 확장하는 방법을 설명한다. MLE에서 교차 엔트로피 손실의 그래디언트를 유도하며, 오차 함수가 볼록하여 전역 최솟값이 유일함을 보인다. SGD를 이용한 온라인 학습과 뉴턴법 기반의 IRLS 알고리즘을 비교하며, IRLS가 가중 최소제곱 문제로 변환되는 과정을 상세히 기술한다. 퍼셉트론 알고리즘을 로지스틱 회귀의 극한 사례로 소개하며 선형 분리 불가능한 데이터에서의 한계를 지적한다. L2 정규화(MAP with 가우시안 사전)와 L1 정규화(MAP with 라플라스 사전)를 통한 과적합 방지를 다루며, 입력 특징의 표준화가 정규화의 공정한 적용에 필요함을 설명한다. 다항 로지스틱 회귀(소프트맥스 회귀)를 K클래스 문제로 확장하고, MLE의 그래디언트와 헤시안을 유도한다. 로버스트 로지스틱 회귀에서 혼합 우도와 바이-템퍼드 손실을 통해 레이블 노이즈에 강건한 학습을 설명한다. 계층적 분류와 대규모 클래스 처리(적응적 소프트맥스, 계층적 소프트맥스)의 실용적 기법을 다룬다. 베이지안 로지스틱 회귀에서 라플라스 근사를 적용하여 가중치의 사후분포를 가우시안으로 근사하고, 예측 분포를 프로빗 근사를 통해 계산한다. 최대 엔트로피 분류기가 로지스틱 회귀와 동치임을 정보이론적 관점에서 보인다.

### Ch 11: Linear Regression (pp. 371-414)

**핵심**: 선형 회귀를 다룬다. 최소제곱, 릿지 회귀, 라소, 탄력적 그물, 로버스트 회귀, 베이지안 선형 회귀, 변수 선택, 혼합효과 모형을 설명한다.

**키워드**: `least squares`, `ridge`, `lasso`, `elastic net`, `Bayesian linear regression`

**상세**: → `Probabilistic Machine Learning An Introduction_azure_full.md` Ch 11 (line 488)

최소제곱 선형 회귀의 정규방정식 w = (XTX)^{-1}XTy를 유도하며, QR 분해와 SVD를 이용한 수치적으로 안정한 계산법을 제시한다. 결정계수 R²를 모형 적합도 측정 지표로 소개하고 잔차 분석의 기본을 다룬다. 릿지 회귀(L2 정규화)를 도입하여 정규방정식에 αI를 추가하는 효과를 보이고, PCA와의 연결을 통해 주성분 방향별 축소 효과를 분석한다. 라소(L1 정규화)가 MAP 추정 with 라플라스 사전에 해당하며, 왜 희소 해를 생성하는지를 기하학적(다이아몬드 제약)·최적화(소프트 임계화) 관점에서 설명한다. 정규화 경로의 시각화, 최소제곱·릿지·라소·부분집합 선택의 비교, 변수 선택 일관성을 다룬다. 그룹 라소와 탄력적 그물(릿지+라소)을 소개하며 상관된 특징 그룹의 처리에 대한 장점을 설명한다. 회귀 스플라인에서 B-스플라인 기저와 평활 스플라인, 일반화 가법 모형(GAM)을 통한 비선형 회귀를 다룬다. 로버스트 선형 회귀에서 라플라스 우도, 스튜던트-t 우도, 후버 손실, RANSAC을 통한 이상치 처리를 설명한다. 베이지안 선형 회귀에서 가우시안 사전 하의 사후분포를 폐쇄형으로 구하고, 예측 분포의 불확실성이 훈련 데이터에서 먼 영역에서 증가함을 보인다. 자동 관련성 결정(ARD)을 개별 가중치에 대한 독립적 사전분포로 소개하여 불필요한 특징을 자동으로 제거하는 기제를 설명한다.

일반화 선형 모형(GLM)의 프레임워크를 지수족 분포의 응답 변수, 선형 예측자 η = wTx, 링크 함수 g(μ) = η의 세 구성요소로 정의한다. 선형 회귀를 항등 링크와 가우시안 분포, 로지스틱 회귀를 로짓 링크와 베르누이 분포로 GLM의 특수한 경우로 통합한다. 포아송 회귀를 로그 링크와 포아송 분포로 정의하여 카운트 데이터의 모델링에 적합한 GLM을 소개한다. 프로빗 회귀를 프로빗 링크 함수(표준 정규 CDF의 역함수)를 사용하는 이진 분류 모형으로 설명하며 로지스틱 회귀와 비교한다. 정준 링크 함수가 지수족의 자연 매개변수와 선형 예측자를 직접 연결하여 계산 및 이론적 편의성을 제공함을 설명한다. 비정준 링크 함수의 사용도 가능하며 도메인 지식에 따라 적절한 링크를 선택할 수 있음을 논한다. MLE에 의한 GLM 매개변수 추정에서 스코어 함수와 피셔 정보를 이용한 반복 가중 최소제곱(IRLS) 또는 경사하강법을 적용한다. 보험 청구 예측 예제를 통해 포아송 GLM의 실제 적용과 해석을 시연한다. GLM이 선형 모형의 자연스러운 확장으로서 다양한 유형의 응답 변수를 통합적으로 처리할 수 있는 유연한 프레임워크임을 요약한다. 과산포(overdispersion) 문제와 음이항 회귀 등의 확장 가능성을 언급한다.

ReLU, 리키 ReLU, ELU, Swish/SiLU, GELU 등 현대적 활성화 함수를 비교하며, 시그모이드/tanh의 기울기 소실 문제를 지적한다. 역전파 알고리즘을 전방 모드와 역방 모드 자동미분으로 설명하며, 계산 그래프를 통해 벡터-야코비안 곱(VJP)의 효율성을 보인다. 학습률 튜닝(웜업, 코사인 감쇠, 1사이클 정책)과 기울기 소실/폭발 문제의 해결책을 다룬다. 잔차 연결(skip connection)이 그래디언트 흐름을 개선하여 매우 깊은 네트워크의 학습을 가능케 하는 원리를 설명한다. 배치 정규화와 레이어 정규화가 내부 공변량 이동을 완화하고 학습을 안정화하는 기제를 논한다. 가중치 초기화 전략(Xavier, He 초기화)이 각 층의 분산을 일정하게 유지하는 원리를 설명한다. 정규화 기법으로 조기 종료, 가중치 감쇠, 드롭아웃(베이지안 근사로서의 해석), 희소 DNN을 소개한다. 베이지안 신경망과 SGD의 암묵적 정규화 효과, 과매개변수화 모형의 이중 하강 현상을 다룬다. 방사 기저함수 네트워크와 전문가 혼합(MoE)을 대안적 피드포워드 아키텍처로 소개하며, 게이팅 네트워크를 통한 조건부 계산의 효율성을 설명한다.

### Ch 14: Neural Networks for Images (pp. 467-502)

**핵심**: 이미지를 위한 신경망을 다룬다. 합성곱 신경망(CNN), 합성곱 연산, 풀링, 주요 CNN 아키텍처(LeNet, AlexNet, ResNet 등), 객체 탐지, 의미론적 분할, 이미지 생성을 설명한다.

**키워드**: `CNN`, `convolution`, `pooling`, `ResNet`, `object detection`, `semantic segmentation`

합성곱 신경망(CNN)의 핵심 연산인 합성곱 층을 도입하며, 국소 수용 영역, 가중치 공유, 등변성의 원리를 설명한다. 합성곱의 수학적 정의(교차상관), 패딩, 스트라이드, 다채널 입출력의 처리를 상세히 다루며 매개변수 수의 효율성을 완전연결층과 비교한다. 풀링 층(최대 풀링, 평균 풀링)이 공간적 불변성과 차원 축소를 제공하며, 스트라이드 합성곱이 풀링의 대안이 될 수 있음을 설명한다. 배치 정규화, 레이어 정규화, 그룹 정규화, 인스턴스 정규화 등 다양한 정규화 층을 비교한다. 주요 CNN 아키텍처의 발전을 LeNet(1998)부터 AlexNet(2012), GoogLeNet/Inception(2014), ResNet(2015), DenseNet(2017)까지 연대순으로 추적한다. 신경 아키텍처 탐색(NAS)이 아키텍처 설계를 자동화하는 방법을 소개한다. 확장 합성곱(dilated convolution), 전치 합성곱(transposed convolution), 깊이별 분리 합성곱(depthwise separable convolution)의 특수 합성곱 변형을 다룬다. 이미지 태깅, 객체 탐지(R-CNN, SSD, YOLO), 인스턴스 분할(Mask R-CNN), 시맨틱 분할(U-Net), 인체 자세 추정 등 판별적 비전 과제에의 응용을 소개한다. CNN의 역전파를 통한 이미지 생성으로 학습된 특징 시각화, Deep Dream, 신경 스타일 전이를 설명한다. CNN이 계층적으로 저수준에서 고수준의 시각 특징을 학습하는 과정을 시각화를 통해 보인다.

### Ch 15: Neural Networks for Sequences (pp. 503-544)

**핵심**: 순차 데이터를 위한 신경망을 다룬다. 순환 신경망(RNN, LSTM, GRU), 1차원 CNN, 어텐션 메커니즘, 트랜스포머, 자기지도학습(BERT, GPT)을 설명한다.

**키워드**: `RNN`, `LSTM`, `GRU`, `attention`, `transformer`, `BERT`, `GPT`

순환 신경망(RNN)을 은닉 상태를 통해 가변 길이 시퀀스를 처리하는 모형으로 정의하고, Vec2Seq(시퀀스 생성), Seq2Vec(시퀀스 분류), Seq2Seq(시퀀스 번역)의 세 패턴을 소개한다. 교사 강제(teacher forcing)를 훈련 시 이전 시점의 정답 출력을 다음 입력으로 사용하는 기법으로 설명하며 노출 편향 문제를 지적한다. 시간을 통한 역전파(BPTT)에서의 기울기 소실·폭발 문제를 분석하고, 기울기 클리핑의 필요성을 설명한다. LSTM이 셀 상태, 망각 게이트, 입력 게이트, 출력 게이트를 통해 장기 의존성을 학습하는 구조를 상세히 기술하며, GRU를 LSTM의 간소화 변형으로 소개한다. 빔 서치를 자기회귀 생성에서 탐욕적 디코딩보다 나은 시퀀스를 찾는 방법으로 설명한다. 1차원 CNN을 시퀀스 분류(TextCNN)와 인과적 시퀀스 생성(WaveNet)에 적용하는 방법을 다룬다. 어텐션 메커니즘을 소프트 사전 탐색으로 도입하며, 쿼리-키-값 구조와 비모수 어텐션(커널 회귀)과의 관계를 설명한다. 트랜스포머 아키텍처를 자기 어텐션, 다중 헤드 어텐션, 위치 인코딩, 피드포워드 층의 구성으로 상세히 기술하며 RNN·CNN과의 비교를 제시한다. 효율적 트랜스포머 변형(희소 어텐션, 저랭크 근사, 선형 어텐션)을 소개하여 O(n²) 복잡도 문제의 해결 방안을 논한다. 언어 모형과 비지도 표현 학습에서 ELMo, BERT(마스크드 언어 모형), GPT(인과적 언어 모형) 등 사전학습-미세조정 패러다임을 설명한다.

### Ch 16: Exemplar-based Methods (pp. 547-566)

**핵심**: 예시 기반 방법을 다룬다. K-최근접이웃(KNN), 거리 측도 학습, 커널 밀도 추정을 설명한다.

**키워드**: `KNN`, `distance metric learning`, `kernel density estimation`

**상세**: → `Probabilistic Machine Learning An Introduction_azure_full.md` Ch 16 (line 97)

K-최근접이웃(KNN) 분류를 도입하여 새 데이터 점을 K개 가장 가까운 훈련 점의 다수결로 분류하는 비모수적 방법을 설명한다. K의 선택이 편향-분산 트레이드오프를 결정하며, K=1일 때 최근접이웃 규칙의 오류율이 최적 베이즈 오류율의 2배 이내임을 소개한다. 차원의 저주가 KNN에 미치는 영향—고차원에서 데이터가 희소해져 이웃 간 거리가 유사해지는 문제—을 분석한다. KD-트리, 국소 민감 해싱(LSH) 등을 통한 속도·메모리 효율화 방법을 소개한다. 거리 측도 학습을 마할라노비스 거리의 행렬 M 학습으로 정의하며, 선형(LMNN, NCA) 및 볼록 방법을 다룬다. 심층 거리 측도 학습(DML)에서 샴 네트워크와 삼중항 손실(앵커-양성-음성), 대조 손실, N-pair 손실 등의 랭킹 손실을 비교한다. 하드 네거티브 마이닝, 세미하드 네거티브 선택, 프록시 앵커 등 학습 효율을 높이는 기법과 임베딩 정규화·온도 스케일링 등의 훈련 기법을 다룬다. 커널 밀도 추정(KDE)을 도입하여 밀도 커널(가우시안, 에파네치니코프)과 파르젠 창 추정기의 수학적 정의를 제시한다. 대역폭 매개변수 h의 선택이 KDE의 매끄러움을 결정하며, 교차검증이나 실버만 규칙 등의 선택 방법을 소개한다. KDE에서 KNN 분류로의 연결을 보이며, 커널 회귀(나다라야-왓슨 추정)를 비모수 회귀 방법으로 설명한다.

### Ch 17: Kernel Methods (pp. 567-602)

**핵심**: 커널 방법을 다룬다. 메르서 커널, 가우시안 과정(GP), 서포트 벡터 머신(SVM), 커널화 방법의 비교를 설명한다.

**키워드**: `Mercer kernels`, `Gaussian processes`, `SVM`, `kernel trick`

메르서 커널의 정의와 메르서 정리를 소개하여 양정치 커널이 암묵적 특성 공간에서의 내적에 해당함을 보인다. 주요 커널 함수—RBF(가우시안), 마테른, 다항식, 코사인 유사도, 문자열 커널—의 성질과 하이퍼파라미터의 역할(길이 척도, 차수)을 비교한다. 가우시안 과정(GP)을 함수에 대한 사전분포로 정의하고, 노이즈 없는·있는 관측 하에서의 예측 분포를 폐쇄형으로 유도한다. GP와 커널 회귀의 관계를 분석하며, 가중치 공간 관점과 함수 공간 관점의 동치성을 보인다. 초매개변수(커널 매개변수, 노이즈 분산)의 한계우도 최대화를 통한 학습과 수치적 안정성(촐레스키 분해, 조깅)을 다룬다. GP 분류에서 비가우시안 우도에 대한 라플라스 근사와 변분 추론의 적용을 설명한다. 신경망과 GP의 연결(무한 폭 극한에서의 수렴)과 신경 접선 커널(NTK)을 소개한다. SVM을 최대 마진 분류기로 도입하며, 쌍대 문제 유도, 서포트 벡터의 희소성, 소프트 마진(슬랙 변수 C), 커널 트릭을 상세히 다룬다. SVM 출력을 확률로 변환하는 플랫 스케일링, 로지스틱 회귀와의 관계(힌지 손실 대 로그 손실), 다중 클래스 확장을 설명한다. 희소 벡터 머신(RVM)과 커널 릿지 회귀, SVM 회귀(ε-SVR)를 포함한 커널 기반 방법의 비교 분석을 제시한다.

### Ch 18: Trees, Forests, Bagging, and Boosting (pp. 603-624)

**핵심**: 트리 기반 앙상블 방법을 다룬다. 결정 트리(CART), 배깅, 랜덤 포레스트, 그래디언트 부스팅(XGBoost, LightGBM), 해석가능성 도구를 설명한다.

**키워드**: `CART`, `random forests`, `gradient boosting`, `XGBoost`, `LightGBM`

**상세**: → `Probabilistic Machine Learning An Introduction_azure_full.md` Ch 18 (line 99)

결정 트리(CART)를 입력 공간의 축 정렬 재귀 분할로 정의하며, 분류(지니 불순도, 교차 엔트로피)와 회귀(MSE)에서의 분할 기준을 설명한다. 탐욕적 트리 성장 후 비용-복잡도 가지치기를 통한 과적합 제어를 다루며, 결정 트리의 장점(해석가능성, 혼합형 특징 처리, 특징 스케일 불변)과 단점(불안정성, 축 정렬 분할의 한계)을 논한다. 배깅(부트스트랩 집계)을 여러 모형의 분산을 줄이는 앙상블 방법으로 소개하며, 각 부트스트랩 표본에 트리를 학습하여 평균(회귀) 또는 투표(분류)하는 과정을 설명한다. 랜덤 포레스트가 배깅에 무작위 특징 부분집합 선택을 추가하여 트리 간 상관을 줄이고 더 나은 일반화를 달성하는 원리를 보인다. 특징 중요도를 순열 중요도와 불순도 감소 기반으로 측정하는 방법을 다룬다. 부스팅의 일반 원리를 순차적 모형 결합으로 소개하며 AdaBoost의 지수 손실 최소화 해석을 제시한다. 그래디언트 부스팅을 함수 공간에서의 경사하강으로 설명하고, 결정 트리를 약한 학습기로 사용하는 그래디언트 부스팅 결정 트리(GBDT)를 다룬다. XGBoost의 정규화된 목적함수(2차 근사, L1/L2 정규화, 축소율)와 효율적 구현(열 블록, 히스토그램 기반 분할)을 설명한다. LightGBM의 리프 우선 성장과 범주형 특징의 직접 처리, CatBoost의 순서 부스팅에 의한 타겟 누출 방지를 비교한다. 부분 의존성 그래프(PDP), SHAP 값 등 해석가능성 도구를 통한 트리 앙상블의 예측 설명을 소개한다.

### Ch 20: Dimensionality Reduction (pp. 657-714)

**핵심**: 차원축소 방법을 다룬다. PCA, 인자 분석, 오토인코더, 다양체 학습(t-SNE, UMAP, Isomap)을 설명한다.

**키워드**: `PCA`, `factor analysis`, `autoencoders`, `t-SNE`, `UMAP`, `manifold learning`

**상세**: → `Probabilistic Machine Learning An Introduction_azure_full.md` Ch 20 (line 106)

주성분 분석(PCA)을 분산 최대화와 재구성 오차 최소화의 두 관점에서 유도하며, 해가 공분산 행렬의 상위 K개 고유벡터임을 보인다. 고유값의 비율로 설명 분산을 측정하여 적절한 차원 수 K를 선택하는 스크리 그래프(elbow method)를 소개한다. 확률적 PCA와 인자 분석을 잠재 변수 모형으로 정의하고, 인자 분석이 관측별 노이즈 분산을 허용하는 점에서 PPCA와 차별됨을 설명한다. EM 알고리즘을 PPCA와 인자 분석의 매개변수 추정에 적용하며, 고차원에서 공분산 행렬의 명시적 계산 없이 작동하는 효율성을 보인다. 인자 회전(바리맥스 등)을 통한 인자 해석성 향상과 인자 수 선택(병렬 분석, BIC)을 다룬다. 오토인코더를 신경망 기반 비선형 차원 축소로 소개하며, 인코더-디코더 구조, 병목(bottleneck) 설계, 디노이징 오토인코더, 변분 오토인코더(VAE)의 기본 개념을 설명한다. 다양체 학습 방법으로 t-SNE가 고차원에서의 확률적 이웃 보존을 통해 2차원 시각화를 수행하며, 스튜던트-t 분포를 사용하여 군중 문제를 해결하는 원리를 설명한다. UMAP이 t-SNE보다 전역 구조를 더 잘 보존하며 계산적으로 효율적인 방법임을 소개한다. Isomap, LLE(국소 선형 임베딩), 라플라시안 고유맵 등의 고전적 다양체 학습 방법을 비교한다. 커널 PCA를 통한 비선형 PCA와 다차원 척도법(MDS)의 연결을 설명한다.

### Ch 21: Clustering (pp. 715-740)

**핵심**: 군집화 방법을 다룬다. K-평균, 가우시안 혼합 모형(GMM)과 EM 알고리즘, 계층적 군집화, 스펙트럼 군집화, 군집 유효성 평가를 설명한다.

**키워드**: `K-means`, `GMM`, `EM algorithm`, `hierarchical clustering`, `spectral clustering`

**상세**: → `Probabilistic Machine Learning An Introduction_azure_full.md` Ch 21 (line 107)

K-평균 군집화를 목적함수(군집 내 제곱합)의 반복 최소화로 정의하며, 할당 단계와 갱신 단계의 수렴을 보인다. K-평균의 초기화 민감성에 대한 해결책으로 K-means++ 초기화와 다중 재시작을 소개한다. K의 선택을 위한 엘보 방법과 실루엣 계수를 논하며, K-medoids(PAM)를 비유클리드 거리에도 적용 가능한 변형으로 소개한다. 가우시안 혼합 모형(GMM)을 확률적 군집화 모형으로 정의하고, EM 알고리즘의 E단계(책임도 계산)와 M단계(매개변수 갱신)를 유도한다. GMM이 K-평균의 소프트 할당 일반화임을 보이며, 공분산 구조(구형, 대각, 완전)에 따른 모형 복잡도와 과적합의 트레이드오프를 논한다. BIC, AIC를 이용한 성분 수 선택과 베이지안 비모수 접근(디리클레 과정 혼합)을 간략히 소개한다. 계층적 군집화를 병합(agglomerative)과 분할(divisive) 방식으로 구분하며, 연결 기준(단일, 완전, 평균, 워드)에 따른 덴드로그램의 차이를 설명한다. 스펙트럼 군집화를 그래프 라플라시안의 고유벡터를 이용한 비볼록 군집화 문제의 이완으로 소개하며, 정규화 컷과의 관계를 논한다. 군집 유효성 평가를 위한 내부 지표(실루엣, 데이비스-볼딘)와 외부 지표(조정 랜드 지수, 정규화 상호정보량)를 다룬다. DBSCAN을 밀도 기반 군집화로 소개하며 임의 형태의 군집 발견과 이상치 탐지 능력을 설명한다.

### Ch 22: Recommender Systems (pp. 741-752)

**핵심**: 추천 시스템을 다룬다. 협업 필터링(행렬 분해, 신경 협업 필터링), 콘텐츠 기반 필터링, 하이브리드 방법을 설명한다.

**키워드**: `collaborative filtering`, `matrix factorization`, `content-based filtering`

추천 시스템의 기본 문제를 사용자-아이템 상호작용 행렬의 결측값 예측으로 정의하고, 명시적 피드백(평점)과 암묵적 피드백(클릭, 구매)을 구분한다. 협업 필터링을 사용자-아이템 유사성에 기반한 접근으로 소개하며, 사용자 기반과 아이템 기반 이웃 방법을 다룬다. 행렬 분해(MF)를 저랭크 근사로 정의하여 사용자와 아이템을 잠재 인자 공간에 임베딩하며, 정규화된 제곱 오차를 SGD 또는 ALS로 최소화하는 학습을 설명한다. 확률적 행렬 분해(PMF)를 가우시안 노이즈 모형 하의 최대우도로 해석하여 정규화의 베이지안 정당성을 제공한다. 편향 항(전역, 사용자, 아이템 편향)의 추가와 암묵적 피드백 통합(SVD++), 시간적 동역학 모형을 다룬다. 신경 협업 필터링(NCF)이 MF의 선형적 상호작용 모델링을 비선형 신경망으로 확장하여 복잡한 사용자-아이템 관계를 포착하는 방법을 소개한다. 콘텐츠 기반 필터링이 아이템의 특성(메타데이터, 텍스트, 이미지)을 이용하여 콜드 스타트 문제를 해결하는 방법을 설명한다. 하이브리드 방법이 협업 필터링과 콘텐츠 기반 방법을 결합하여 각각의 한계를 보완하는 접근을 다룬다. 평가 지표로 RMSE, 정밀도@K, 재현율@K, NDCG를 소개하며 오프라인 평가와 온라인 A/B 테스트의 차이를 논한다. 탐색-활용 트레이드오프와 다양성·공정성 등 추천 시스템의 실용적 과제를 언급한다.

### Ch 23: Graph Embeddings (pp. 753-772)

**핵심**: 그래프 임베딩 방법을 다룬다. 그래프 신경망(GNN), 노드/그래프 수준 임베딩, 지식 그래프 임베딩을 설명한다.

**키워드**: `GNN`, `graph embeddings`, `knowledge graphs`, `node embeddings`

**상세**: → `Probabilistic Machine Learning An Introduction_azure_full.md` Ch 23 (line 109)

그래프 임베딩의 목표를 그래프의 노드, 에지, 서브그래프를 저차원 벡터로 표현하여 하류 과제에 활용하는 것으로 정의한다. 그래프 신경망(GNN)의 기본 원리로 이웃 집계(neighborhood aggregation)를 소개하며, 각 노드가 이웃의 특징을 반복적으로 수집·변환하여 표현을 갱신하는 메시지 전달 프레임워크를 설명한다. 그래프 합성곱 네트워크(GCN)가 스펙트럼 그래프 이론의 1차 근사로 유도되며, 정규화된 인접 행렬을 사용한 층별 전파 규칙을 제시한다. GraphSAGE의 샘플링-집계 방식이 대규모 그래프에서의 확장성을 제공하며, 다양한 집계 함수(평균, LSTM, 풀링)를 사용하는 방법을 다룬다. 그래프 어텐션 네트워크(GAT)가 이웃에 대한 어텐션 가중치를 학습하여 이질적 이웃 구조를 처리하는 방법을 설명한다. 노드 수준 임베딩에서 DeepWalk와 Node2Vec가 랜덤 워크를 이용한 비지도 학습으로 노드를 임베딩하는 방법을 소개한다. 그래프 수준 임베딩에서 풀링(합, 평균, 계층적 풀링)을 통해 노드 표현을 그래프 표현으로 집계하는 방법을 다룬다. 지식 그래프 임베딩에서 엔티티와 관계를 저차원 공간에 표현하는 TransE, DistMult, ComplEx 등의 모형을 소개하며 링크 예측에의 활용을 설명한다. GNN의 응용으로 분자 특성 예측, 소셜 네트워크 분석, 추천 시스템, 조합 최적화 등을 언급한다. 과도한 평활화(over-smoothing) 문제와 이종 그래프·동적 그래프에 대한 확장을 간략히 논한다.
